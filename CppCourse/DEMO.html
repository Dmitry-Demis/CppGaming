✨ Основные возможности

💻 Онлайн компилятор C++

Встроенный компилятор с поддержкой GCC (13.2, 12.2, 11.4) и Clang (16.0, 15.0, 14.0). Выбор архитектуры (x86-64, x86) и стандарта C++ (C++11 до C++23). Использует API Compiler Explorer (godbolt.org).

🎬 GSAP Анимации

Плавные анимации элементов при загрузке и прокрутке страницы. ScrollTrigger для анимации при появлении в viewport. Elastic, back.out и другие easing функции.

📝 Brace-init инициализация

Все примеры кода используют современный синтаксис C++11+: int x{42}; вместо int x = 42;

🎮 Интерактивные квизы

Система тестирования знаний с прогресс-баром, обратной связью и подробными объяснениями к каждому вопросу.

🏆 Система достижений

10 достижений для разблокировки: от "Первая программа" до "Перфекционист". Векторные SVG иконки с анимацией разблокировки.

📊 Интерактивные диаграммы

SVG-диаграммы типов данных с древовидной структурой. Цветовое кодирование для различных категорий типов.

📚 Содержание курса

  • Введение в фундаментальные типы данных
  • Целочисленные типы (short, int, long, long long)
  • Типы с плавающей точкой (float, double, long double)
  • Символьные типы (char, wchar_t, char16_t, char32_t)
  • Логический тип (bool)
  • Использование std::numeric_limits
  • ISO C++ цитаты и ссылки на стандарт

🎯 Особенности реализации

  • Адаптивный дизайн для всех устройств
  • Тёмная тема с голубо-фиолетовыми акцентами
  • Подсветка синтаксиса кода (Prism.js)
  • Копирование кода одним кликом
  • Поддержка Tab в редакторе кода
  • Отображение ошибок компиляции
  • Вывод результатов выполнения программы
  • Сохранение прогресса в localStorage
