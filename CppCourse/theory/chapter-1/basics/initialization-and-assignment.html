    Инициализация и присваивание

    Два похожих, но разных понятия: инициализация задаёт начальное значение при создании переменной, а присваивание меняет значение уже существующей переменной.

    Способы инициализации

    В C++ есть несколько синтаксисов инициализации:

    C++ — инициализация переменных 
    // загрузка...
    Виды инициализации
    • Copy initialization: int x = 5; — классический стиль C
    • Direct initialization: int x(5); — вызов конструктора
    • List initialization C++11: int x{5}; — предпочтительный современный стиль
    • Value initialization: int x{}; — инициализация нулём

    Преимущество list initialization

    Фигурные скобки {} запрещают сужающие преобразования (narrowing conversions) — когда значение не помещается в тип. Это помогает поймать ошибки на этапе компиляции.

    Narrowing conversion

    int x = 3.14; — компилируется, но теряет дробную часть (x = 3).
    int x{3.14}; — ошибка компиляции. Это защита от случайных потерь данных.

    Присваивание

    Оператор = копирует значение справа в переменную слева. Это не математическое равенство — это команда «записать значение».

    = vs ==
    • x = 5 — присваивание: записать 5 в x
    • x == 5 — сравнение: проверить, равно ли x пяти
