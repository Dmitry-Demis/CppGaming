    Глобальный генератор случайных чисел (Random.h)

    Когда генератор случайных чисел нужен в нескольких функциях или файлах, удобно иметь один глобальный объект. Это один из редких случаев, когда глобальная переменная оправдана — она уникальна в программе и используется повсеместно.

    Проблема: генератор в каждой функции

    Создавать std::mt19937 в каждой функции — плохая идея: это медленно, и каждый генератор будет давать независимые (возможно, коррелированные) последовательности.

    Готовое решение: Random.h

    Вот заголовочный файл, который создаёт один глобальный генератор, доступный из любого файла. Требует C++17.

    C++ — Random.h (C++17) 
    #ifndef RANDOM_MT_H
#define RANDOM_MT_H

#include <chrono>
#include <random>

namespace Random
{
    // Создаёт и возвращает хорошо инициализированный std::mt19937
    inline std::mt19937 generate()
    {
        std::random_device rd {};
        std::seed_seq ss {
            static_cast<std::seed_seq::result_type>(
                std::chrono::steady_clock::now().time_since_epoch().count()),
            rd(), rd(), rd(), rd(), rd(), rd(), rd()
        };
        return std::mt19937 { ss };
    }

    // Глобальный объект — один на всю программу (inline = нет нарушения ODR)
    inline std::mt19937 mt { generate() };

    // Генерирует случайное int в диапазоне [min, max] включительно
    inline int get(int min, int max)
    {
        return std::uniform_int_distribution { min, max }(mt);
    }

    // Шаблонная версия для других целочисленных типов
    template <typename T>
    T get(T min, T max)
    {
        return std::uniform_int_distribution<T> { min, max }(mt);
    }

    // Версия с явным указанием типа результата
    template <typename R, typename S, typename T>
    R get(S min, T max)
    {
        return get<R>(static_cast<R>(min), static_cast<R>(max));
    }
}

#endif

    Использование

    C++ — использование Random.h 
    #include "Random.h"
#include <iostream>

int main()
{
    // Случайное int от 1 до 6
    std::cout << Random::get(1, 6) << '\n';

    // Случайное unsigned int от 1 до 6
    std::cout << Random::get(1u, 6u) << '\n';

    // Явное указание типа результата
    std::cout << Random::get<std::size_t>(0, 100) << '\n';

    // Прямой доступ к генератору для своих распределений
    std::uniform_int_distribution die20 { 1, 20 };
    std::cout << die20(Random::mt) << '\n';

    return 0;
}

    Как это работает

    🔧 Технические детали
    • inline std::mt19937 mt — глобальная переменная с внешней компоновкой. Ключевое слово inline позволяет включать заголовок в несколько .cpp файлов без нарушения ODR.
    • Функция generate() инициализирует генератор комбинацией системных часов и std::random_device — это даёт хорошее качество seed.
    • Генератор создаётся один раз при запуске программы и живёт до её завершения.

    Три шага для использования

    📋 Инструкция
    1. Скопируйте код Random.h в файл Random.h в директории проекта
    2. Добавьте #include "Random.h" в любой .cpp файл, которому нужны случайные числа
    3. Вызывайте Random::get(min, max) — никакой дополнительной настройки не нужно
    Требования

    Random.h требует C++17 из-за использования inline переменных. Убедитесь, что компилятор настроен на стандарт C++17 или новее.
