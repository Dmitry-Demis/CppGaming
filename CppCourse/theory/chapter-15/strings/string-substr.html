    Строки

    Подстроки и проверка начала/конца

    Часто нужно извлечь часть строки или проверить, начинается ли она с определённого префикса. std::string предоставляет удобные методы для этого.

    Извлечение подстроки: substr()

    Метод substr(pos, count) возвращает новую строку — копию count символов начиная с позиции pos.

    C++ — substr() 
    #include <iostream>
#include <string>

int main()
{
    std::string email { "tom@localhost.com" };

    // substr(pos, count): 9 символов начиная с позиции 4
    std::string host { email.substr(4, 9) };
    std::cout << host << '\n'; // localhost

    // Если count больше оставшихся символов — берём всё до конца
    std::string domain { email.substr(4) }; // от позиции 4 до конца
    std::cout << domain << '\n'; // localhost.com

    // Исключение при неверном pos:
    // email.substr(100); // бросает std::out_of_range

    return 0;
}

    Проверка начала и конца строки

    До C++20 для проверки префикса/суффикса использовали compare() или substr(). C++20 добавил удобные методы starts_with() и ends_with().

    C++ — starts_with / ends_with (C++20) 
    #include <iostream>
#include <string>

int main()
{
    std::string url { "https://example.com/page" };

    // C++20: starts_with / ends_with
    if (url.starts_with("https://"))
        std::cout << "Защищённое соединение\n";

    if (url.ends_with("/page"))
        std::cout << "Это страница /page\n";

    // До C++20 — через substr:
    std::string prefix { "https://" };
    if (url.substr(0, prefix.length()) == prefix)
        std::cout << "Тоже работает\n";

    return 0;
}

    Практический пример: парсинг email

    C++ — извлечение частей email 
    #include <iostream>
#include <string>

int main()
{
    std::string email { "user@example.com" };

    // Находим позицию @
    std::size_t at { email.find('@') };

    if (at != std::string::npos) {
        std::string user   { email.substr(0, at) };       // до @
        std::string domain { email.substr(at + 1) };      // после @

        std::cout << "Пользователь: " << user   << '\n'; // user
        std::cout << "Домен: "        << domain << '\n'; // example.com
    }

    return 0;
}
