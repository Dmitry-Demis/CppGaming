    Символьные массивы и C-строки

    Символьный массив (char[]) — это низкоуровневый способ хранить строки в C++, унаследованный из C. В современном C++ для строк используют std::string, но понимание C-строк важно для работы с legacy-кодом и системными API.

    Нулевой терминатор

    Ключевое отличие C-строки от обычного массива символов — нулевой терминатор '\0' в конце. Без него std::cout будет читать память за пределами массива.

    C++ — важность нулевого терминатора 
    #include <iostream>

int main()
{
    char hello1[] { 'h', 'e', 'l', 'l', 'o' };       // 5 элементов, БЕЗ '\0'
    char hello2[] { 'h', 'e', 'l', 'l', 'o', '\0' }; // 6 элементов, с '\0'
    char hello3[] { "hello" };                         // 6 элементов, '\0' добавляется автоматически

    // std::cout << hello1 << '\n'; // ОПАСНО: читает за пределами массива!
    std::cout << hello2 << '\n'; // hello
    std::cout << hello3 << '\n'; // hello

    return 0;
}

    Двумерные символьные массивы (массив строк)

    C++ — массив строк 
    #include <iostream>

int main()
{
    const int max_length { 50 }; // максимальная длина строки (включая '\0')
    char langs[][max_length]
    {
        "C++", "C#", "Python", "Java",
        "Kotlin", "Go", "Dart", "PHP"
    };

    std::cout << langs[0] << '\n'; // C++
    std::cout << langs[2] << '\n'; // Python

    // Перебор всех строк
    for (auto lang : langs)
        std::cout << lang << '\n';

    return 0;
}

    Ввод строки с консоли

    C++ — cin.getline для C-строк 
    #include <iostream>

int main()
{
    const int max_length { 100 };
    char text[max_length] {};

    std::cout << "Введите текст: ";
    std::cin.getline(text, max_length); // читает до '\n', включая пробелы

    std::cout << "Вы ввели: " << text << '\n';

    // Вариант с кастомным разделителем:
    // std::cin.getline(text, max_length, '!'); // читает до '!'

    return 0;
}
    💡 В современном C++ используйте std::string

    Для работы со строками в современном C++ используйте std::string и std::getline(std::cin, str). C-строки нужны только при работе с C API или legacy-кодом.

    #include <string>
std::string s;
std::getline(std::cin, s); // безопасно, без ограничений на длину
