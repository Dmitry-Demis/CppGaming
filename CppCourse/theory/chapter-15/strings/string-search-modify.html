    Строки

    Поиск и изменение строк

    std::string предоставляет богатый набор методов для поиска подстрок, замены, вставки и удаления символов.

    Поиск: find и rfind

    C++ — поиск подстроки
    #include <iostream>
#include <string>

int main()
{
    std::string text { "A friend in need is a friend indeed." };

    // find: первое вхождение
    size_t pos = text.find("friend");
    std::cout << pos << '\n'; // 2

    // find с начальной позицией
    pos = text.find("friend", 10); // ищем с позиции 10
    std::cout << pos << '\n'; // 22

    // rfind: последнее вхождение
    pos = text.rfind("friend");
    std::cout << pos << '\n'; // 22

    // Если не найдено — возвращает std::string::npos
    if (text.find("apple") == std::string::npos)
        std::cout << "Не найдено\n";

    return 0;
}

    Изменение: replace, insert, erase, append

    C++ — изменение строки
    #include <iostream>
#include <string>

int main()
{
    std::string text { "Lang: Java" };

    // replace: заменяем 4 символа с позиции 6 на "C++"
    text.replace(6, 4, "C++");
    std::cout << text << '\n'; // Lang: C++

    // insert: вставляем строку с позиции 5
    text.insert(5, " (best)");
    std::cout << text << '\n'; // Lang: (best) C++

    // erase: удаляем 7 символов с позиции 5
    text.erase(5, 7);
    std::cout << text << '\n'; // Lang: C++

    // append: добавляем в конец
    text.append(" is great!");
    std::cout << text << '\n'; // Lang: C++ is great!

    // substr: извлекаем подстроку
    std::string sub = text.substr(6, 3); // 3 символа с позиции 6
    std::cout << sub << '\n'; // C++

    return 0;
}

    Живое применение: замена всех вхождений

    C++ — замена всех вхождений подстроки
    #include <iostream>
#include <string>

void replaceAll(std::string& text, const std::string& from, const std::string& to)
{
    size_t pos { text.find(from) };
    while (pos != std::string::npos)
    {
        text.replace(pos, from.length(), to);
        pos = text.find(from, pos + to.length());
    }
}

int main()
{
    std::string msg { "Message to Tom: Hello, Tom!" };
    replaceAll(msg, "Tom", "Alice");
    std::cout << msg << '\n'; // Message to Alice: Hello, Alice!
    return 0;
}

    Поиск символов из набора

    C++ — find_first_of, find_last_of
    std::string phone { "Phone: +23415678901" };
std::string digits { "0123456789" };

size_t first = phone.find_first_of(digits); // 8 — первая цифра
size_t last  = phone.find_last_of(digits);  // 18 — последняя цифра

// Извлекаем номер:
std::string number = phone.substr(first, last - first + 1);
std::cout << number << '\n'; // 23415678901
