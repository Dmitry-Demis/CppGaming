    Массивы и std::vector

    Индексация массива через перечисления

    Числовые индексы не говорят ничего о смысле элемента. testScores[2] — чей это результат? Непонятно. Перечисления (enum) решают эту проблему: они дают индексам имена.

    Проблема числовых индексов

    C++ — непонятные числовые индексы 
    std::vector testScores { 78, 94, 66, 77, 14 };
testScores[2] = 76; // кто это? непонятно

    Решение: enum как индексы

    Нескопированные перечисления (enum) неявно конвертируются в std::size_t, поэтому их можно использовать как индексы напрямую:

    C++ — enum как индексы массива 
    #include <cassert>
#include <iostream>
#include <vector>

namespace Students
{
    enum Names
    {
        kenny,    // 0
        kyle,     // 1
        stan,     // 2
        butters,  // 3
        cartman,  // 4
        max_students // 5 — счётчик
    };
}

int main()
{
    // Создаём вектор нужной длины через count-перечислитель
    std::vector<int> testScores(Students::max_students);

    // Теперь индексы говорящие!
    testScores[Students::stan]    = 76;
    testScores[Students::cartman] = 92;
    testScores[Students::kenny]   = 55;

    // Проверяем, что размер совпадает с количеством студентов
    assert(std::size(testScores) == Students::max_students);

    std::cout << "Оценка Стэна: " << testScores[Students::stan] << '\n'; // 76
    std::cout << "Студентов: "    << Students::max_students << '\n';      // 5

    return 0;
}

    Count-перечислитель

    Последний элемент max_students — это count-перечислитель. Его значение автоматически равно количеству предыдущих перечислителей. Если добавить нового студента — размер массива обновится сам:

    C++ — добавление нового студента 
    namespace Students
{
    enum Names
    {
        kenny, kyle, stan, butters, cartman,
        wendy,       // добавили нового студента
        max_students // теперь автоматически = 6
    };
}

// std::vector<int> testScores(Students::max_students); — теперь 6 элементов
// Не нужно менять ничего кроме enum!

    Инициализация списком + assert

    C++ — инициализация + проверка размера 
    enum StudentNames { kenny, kyle, stan, butters, cartman, max_students };

std::vector testScores { 78, 94, 66, 77, 14 };

// Убеждаемся, что количество оценок совпадает с количеством студентов
assert(std::size(testScores) == max_students);
// Если добавить студента в enum, но забыть добавить оценку — assert сработает
    ⚠️ enum class и индексы

    Скопированные перечисления (enum class) не конвертируются в целые неявно. Для них нужен static_cast или перегрузка оператора +. Поэтому для индексации массивов удобнее обычный enum внутри namespace.
