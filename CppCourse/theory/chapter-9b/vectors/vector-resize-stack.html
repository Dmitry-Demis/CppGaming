    Массивы и std::vector

    Изменение размера вектора и стек-поведение

    Главное преимущество std::vector перед другими массивами — он умеет менять размер во время выполнения программы. Это делает его динамическим массивом.

    Длина и ёмкость

    У вектора два разных числа: длина (сколько элементов используется) и ёмкость (сколько памяти выделено).

    C++ — resize, capacity, reserve 
    #include <iostream>
#include <vector>

int main()
{
    std::vector v { 0, 1, 2 }; // length=3, capacity=3
    std::cout << v.size() << ' ' << v.capacity() << '\n'; // 3 3

    v.resize(5); // length=5, capacity=5 (реаллокация)
    std::cout << v.size() << ' ' << v.capacity() << '\n'; // 5 5

    v.resize(3); // length=3, capacity=5 (реаллокации нет!)
    std::cout << v.size() << ' ' << v.capacity() << '\n'; // 3 5

    // reserve: меняет только ёмкость, не длину
    std::vector<int> stack {};
    stack.reserve(10); // выделяем память для 10 элементов
    std::cout << stack.size() << ' ' << stack.capacity() << '\n'; // 0 10

    return 0;
}

    Стек-поведение: push_back и pop_back

    std::vector поддерживает операции стека (LIFO — последний вошёл, первый вышел):

    C++ — push_back, pop_back, back 
    #include <iostream>
#include <vector>

int main()
{
    std::vector<int> stack {};
    stack.reserve(4); // избегаем лишних реаллокаций

    stack.push_back(1); // [1]
    stack.push_back(2); // [1, 2]
    stack.push_back(3); // [1, 2, 3]

    std::cout << "Верхний: " << stack.back() << '\n'; // 3

    stack.pop_back(); // [1, 2]
    stack.pop_back(); // [1]

    std::cout << "Размер: " << stack.size() << '\n'; // 1
    return 0;
}

    Живое применение: ввод неизвестного числа элементов

    C++ — сбор оценок от пользователя 
    #include <iostream>
#include <vector>

int main()
{
    std::vector<int> scores {};

    while (true)
    {
        std::cout << "Введите оценку (-1 для завершения): ";
        int x {};
        std::cin >> x;
        if (x == -1) break;
        scores.push_back(x); // добавляем в конец
    }

    std::cout << "Оценки: ";
    for (const auto& s : scores)
        std::cout << s << ' ';
    std::cout << '\n';
    std::cout << "Всего: " << scores.size() << '\n';

    return 0;
}
    📋 resize vs reserve
    • resize(n) — меняет длину (и ёмкость если нужно). Используйте при доступе через индексы.
    • reserve(n) — меняет только ёмкость. Используйте при добавлении через push_back.
