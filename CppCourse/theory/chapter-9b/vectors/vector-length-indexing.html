    Массивы и std::vector

    Длина вектора и проблема знаковости индексов

    Получить длину вектора просто. Но есть нюанс: стандартная библиотека использует беззнаковые типы для длины и индексов, что создаёт неожиданные проблемы.

    Получение длины

    C++ — три способа получить длину 
    #include <iostream>
#include <vector>

int main()
{
    std::vector primes { 2, 3, 5, 7, 11 };

    // 1. Метод size() — возвращает std::size_t (беззнаковый)
    std::cout << primes.size() << '\n'; // 5

    // 2. std::size() — то же самое (C++17)
    std::cout << std::size(primes) << '\n'; // 5

    // 3. std::ssize() — возвращает знаковый тип (C++20, рекомендуется)
    std::cout << std::ssize(primes) << '\n'; // 5

    // Сохранить в переменную:
    int len { static_cast<int>(primes.size()) }; // явное приведение
    auto len2 { std::ssize(primes) };             // C++20: auto выводит знаковый тип

    return 0;
}

    Проблема знаковости: историческая ошибка

    Дизайнеры стандартной библиотеки C++ в 1997 году выбрали беззнаковые типы для индексов. Сегодня это считается ошибкой. Вот почему это создаёт проблемы:

    C++ — ловушка с беззнаковым индексом 
    #include <iostream>
#include <vector>

int main()
{
    std::vector arr { 4, 6, 7, 3, 8 };

    // Обход в обратном порядке — ОПАСНО с беззнаковым индексом!
    for (std::size_t i { arr.size() - 1 }; i >= 0; --i) // бесконечный цикл!
    {
        std::cout << arr[i] << ' ';
        // Когда i == 0 и мы делаем --i, беззнаковый 0 оборачивается в огромное число!
    }

    return 0;
}

    Решения

    C++ — безопасный обратный обход 
    #include <iostream>
#include <vector>

int main()
{
    std::vector arr { 4, 6, 7, 3, 8 };

    // Вариант 1: знаковый индекс + static_cast (до C++20)
    for (int i { static_cast<int>(arr.size()) - 1 }; i >= 0; --i)
        std::cout << arr[static_cast<std::size_t>(i)] << ' ';
    std::cout << '\n';

    // Вариант 2: std::ssize() (C++20, чище)
    for (auto i { std::ssize(arr) - 1 }; i >= 0; --i)
        std::cout << arr.data()[i] << ' '; // data() принимает знаковый индекс
    std::cout << '\n';

    // Вариант 3 (лучший): range-for в обратном порядке (C++20)
    // — рассмотрим в следующем параграфе

    return 0;
}
    💡 Практический совет

    Если используете индексы — предпочитайте std::ssize() (C++20) или static_cast<int>(v.size()). Лучше всего — избегать индексов вообще и использовать range-for.
