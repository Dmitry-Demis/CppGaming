    Передача и возврат std::array

    Поскольку размер std::array является частью его типа, передача массива в функцию требует особого подхода. Рассмотрим все варианты.

    Передача по значению

    Передача по значению создаёт полную копию массива. Для больших массивов это дорого. Кроме того, функция привязана к конкретному типу и размеру.

    C++ — по значению (не рекомендуется для больших массивов) 
    // загрузка...

    Передача по const-ссылке

    Предпочтительный способ для чтения — передача по const-ссылке. Копирования нет, массив защищён от изменений.

    C++ — по const-ссылке 
    // загрузка...

    Шаблонная функция для любого размера

    Чтобы функция работала с массивами любого типа и размера, используй шаблон. Это самый гибкий и идиоматичный подход.

    C++ — шаблонная функция 
    // загрузка...

    Возврат std::array из функции

    std::array можно возвращать по значению — компилятор применит оптимизацию (NRVO/RVO) и избежит лишнего копирования.

    C++ — возврат массива 
    // загрузка...
    Итог: как передавать std::array

    Для чтения — const std::array<T,N>& или шаблонная функция.
    Для изменения — std::array<T,N>&.
    Возврат по значению — нормально, компилятор оптимизирует.
