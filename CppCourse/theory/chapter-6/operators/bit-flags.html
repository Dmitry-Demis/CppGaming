    Битовые флаги и std::bitset

    Обычно переменная хранит одно значение. Но можно использовать каждый бит переменной как независимый булев флаг. Это называется битовыми флагами и позволяет хранить до 8 булевых значений в одном байте.

    Нумерация битов

    Биты нумеруются справа налево, начиная с 0:

    Позиции битов
    7 6 5 4 3 2 1 0   ← позиция
0 0 0 0 0 1 0 1   ← значение (биты 0 и 2 установлены)

    std::bitset

    Самый удобный способ работы с битовыми флагами — std::bitset из заголовка <bitset>. Он предоставляет четыре ключевых метода:

    Методы std::bitset
    Метод Действие
    test(pos)Проверить бит на позиции pos (0 или 1)
    set(pos)Установить бит в 1 (ничего не делает, если уже 1)
    reset(pos)Сбросить бит в 0 (ничего не делает, если уже 0)
    flip(pos)Инвертировать бит
    C++ — основы std::bitset 
    #include <bitset>
#include <iostream>

int main()
{
    std::bitset<8> bits{ 0b0000'0101 }; // биты 0 и 2 установлены
    bits.set(3);   // установить бит 3 → 0000 1101
    bits.flip(4);  // инвертировать бит 4 → 0001 1101
    bits.reset(4); // сбросить бит 4 → 0000 1101

    std::cout << "Все биты: " << bits << '\n';
    std::cout << "Бит 3: " << bits.test(3) << '\n'; // 1
    std::cout << "Бит 4: " << bits.test(4) << '\n'; // 0

    return 0;
}

    Именованные флаги

    Давайте флагам осмысленные имена — это делает код читаемым:

    C++ — именованные битовые флаги 
    #include <bitset>
#include <iostream>

int main()
{
    constexpr int isHungry   { 0 };
    constexpr int isSad      { 1 };
    constexpr int isHappy    { 3 };
    constexpr int isLaughing { 4 };

    std::bitset<8> me{ 0b0000'0101 };
    me.set(isHappy);
    me.flip(isLaughing);
    me.reset(isLaughing);

    std::cout << "Я счастлив: " << me.test(isHappy) << '\n';
    std::cout << "Я смеюсь: "  << me.test(isLaughing) << '\n';

    return 0;
}

    Дополнительные методы запроса

    C++ — size, count, all, any, none 
    #include <bitset>
#include <iostream>

int main()
{
    std::bitset<8> bits{ 0b0000'1101 };
    std::cout << bits.size()  << " бит в bitset\n";   // 8
    std::cout << bits.count() << " бита установлено\n"; // 3

    std::cout << std::boolalpha;
    std::cout << "Все: "  << bits.all()  << '\n'; // false
    std::cout << "Есть:  " << bits.any()  << '\n'; // true
    std::cout << "Нет:   " << bits.none() << '\n'; // false

    return 0;
}
    Размер std::bitset

    std::bitset оптимизирован для скорости, а не для экономии памяти. std::bitset<8> обычно занимает 4 или 8 байт (до ближайшего sizeof(size_t)), а не 1 байт. Используйте его для удобства, а не для экономии памяти.
