    Побитовые операторы

    Побитовые операторы работают непосредственно с битами целочисленных значений. В отличие от логических операторов, они применяются к каждой паре битов операндов.

    Лучшая практика

    Используйте побитовые операторы с беззнаковыми целыми типами или std::bitset, чтобы избежать неожиданного поведения со знаковыми типами.

    Таблица операторов

    Шесть побитовых операторов
    Оператор Символ Форма Результат
    Сдвиг влево<<x << nБиты x сдвинуты влево на n позиций, новые биты = 0
    Сдвиг вправо>>x >> nБиты x сдвинуты вправо на n позиций, новые биты = 0
    Побитовое НЕ~~xКаждый бит x инвертирован
    Побитовое И&x & yБит = 1, если оба соответствующих бита = 1
    Побитовое ИЛИ|x | yБит = 1, если хотя бы один из битов = 1
    Побитовое XOR^x ^ yБит = 1, если соответствующие биты различаются

    Сдвиги

    Оператор << сдвигает биты влево — биты, вышедшие за пределы, теряются, а справа добавляются нули. Оператор >> работает симметрично вправо.

    Примеры сдвигов (4-битные значения)
    0011 << 1  →  0110
0011 << 2  →  1100
0011 << 3  →  1000  // бит 1 потерян!

1100 >> 1  →  0110
1100 >> 2  →  0011
1100 >> 3  →  0001  // бит потерян!
    C++ — побитовые сдвиги 
    #include <bitset>
#include <iostream>

int main()
{
    std::bitset<4> x { 0b1100 };

    std::cout << x << '\n';         // 1100
    std::cout << (x >> 1) << '\n'; // 0110
    std::cout << (x << 1) << '\n'; // 1000 (бит вышел за пределы)

    return 0;
}
    operator<< и std::cout

    Тот же символ << используется для вывода в поток. Компилятор определяет смысл по типу левого операнда: если это поток — вывод, если целое — сдвиг. При использовании сдвига в выражении с std::cout обязательно ставьте скобки: std::cout << (x << 1).

    Побитовое НЕ (~)

    Оператор ~ инвертирует каждый бит. Результат зависит от ширины типа: ~0b0100 для 4-битного значения даёт 1011, а для 8-битного — 1111 1011.

    C++ — побитовое НЕ 
    #include <bitset>
#include <iostream>

int main()
{
    std::bitset<4> b4 { 0b0100 }; // 0100
    std::bitset<8> b8 { 0b0100 }; // 0000 0100

    std::cout << ~b4 << '\n'; // 1011  → значение 11
    std::cout << ~b8 << '\n'; // 1111 1011 → значение 251

    return 0;
}

    Побитовое ИЛИ (|)

    Для каждой пары битов: результат = 1, если хотя бы один бит = 1.

    Пример: 0101 | 0110
    0 1 0 1  OR
0 1 1 0
-------
0 1 1 1   →  0111
    C++ — побитовое ИЛИ 
    #include <bitset>
#include <iostream>

int main()
{
    std::cout << (std::bitset<4>{ 0b0101 } | std::bitset<4>{ 0b0110 }) << '\n'; // 0111
    return 0;
}

    Побитовое И (&)

    Для каждой пары битов: результат = 1, только если оба бита = 1.

    Пример: 0101 & 0110
    0 1 0 1  AND
0 1 1 0
--------
0 1 0 0   →  0100
    C++ — побитовое И 
    #include <bitset>
#include <iostream>

int main()
{
    std::cout << (std::bitset<4>{ 0b0101 } & std::bitset<4>{ 0b0110 }) << '\n'; // 0100
    return 0;
}

    Побитовое XOR (^)

    Для каждой пары битов: результат = 1, если биты различаются.

    Пример: 0110 ^ 0011
    0 1 1 0  XOR
0 0 1 1
-------
0 1 0 1   →  0101

    Операторы присваивания

    Побитовые операторы присваивания
    Оператор Эквивалент
    x <<= nx = x << n
    x >>= nx = x >> n
    x &= yx = x & y
    x |= yx = x | y
    x ^= yx = x ^ y
    C++ — оператор присваивания со сдвигом 
    #include <bitset>
#include <iostream>

int main()
{
    std::bitset<4> bits { 0b0100 };
    bits >>= 1;
    std::cout << bits << '\n'; // 0010
    return 0;
}
    Интегральное повышение типа

    Если операнды побитовых операторов имеют тип меньше int, они автоматически повышаются до int или unsigned int. Операторы ~ и << чувствительны к ширине типа и могут дать неожиданный результат. Используйте static_cast для приведения обратно к нужному типу.

    Правила вычисления по столбцам

    Итог
    • OR: если хотя бы один бит в столбце = 1 → результат 1
    • AND: если все биты в столбце = 1 → результат 1
    • XOR: если нечётное количество единиц в столбце → результат 1
