    Шаблоны классов

    Как шаблон функции позволяет создавать функции для разных типов, шаблон класса позволяет создавать типы (struct/class) для разных типов данных.

    Проблема без шаблонов

    Без шаблонов пришлось бы дублировать структуру для каждого типа:

    C++ — дублирование
    struct PairInt    { int first{};    int second{};    };
struct PairDouble { double first{}; double second{}; };
// ... и так для каждого типа

    Определение шаблона класса

    C++
    #include <iostream>

template <typename T>
struct Pair
{
    T first{};
    T second{};
};

int main()
{
    Pair<int>    p1 { 5, 6 };        // Pair<int>
    Pair<double> p2 { 1.2, 3.4 };   // Pair<double>
    Pair<double> p3 { 7.8, 9.0 };   // использует уже созданный Pair<double>

    std::cout << p1.first << ' ' << p1.second << '\n';
    return 0;
}

    Шаблон функции для шаблона класса

    C++
    template <typename T>
struct Pair { T first{}; T second{}; };

template <typename T>
constexpr T max(Pair<T> p)
{
    return (p.first < p.second ? p.second : p.first);
}

int main()
{
    Pair<int> p1 { 5, 6 };
    std::cout << max(p1) << '\n'; // 6 — тип выводится автоматически
    return 0;
}

    Несколько параметров типа

    C++
    template <typename T, typename U>
struct Pair { T first{}; U second{}; };

int main()
{
    Pair<int, double> p1 { 1, 2.3 };
    Pair<double, int> p2 { 4.5, 6 };
    Pair<int, int>    p3 { 7, 8 };
    return 0;
}

    std::pair

    В стандартной библиотеке уже есть std::pair из заголовка <utility>:

    C++
    #include <iostream>
#include <utility>

int main()
{
    std::pair<int, double> p { 1, 2.3 };
    std::cout << p.first << ' ' << p.second << '\n';
    return 0;
}
    Лучшая практика

    В реальном коде используйте std::pair вместо собственной реализации.
