    Перегрузка функций

    Введение в перегрузку функций

    Представьте: вам нужна функция print, которая умеет выводить и int, и double, и std::string. Без перегрузки пришлось бы писать printInt, printDouble, printString... Перегрузка функций позволяет дать нескольким функциям одно имя, если они отличаются типами параметров.

    Проблема без перегрузки

    C++ — без перегрузки: много имён 
    int addInt(int x, int y)       { return x + y; }
double addDouble(double x, double y) { return x + y; }
// Нужно помнить разные имена, легко ошибиться

    Перегрузка: одно имя, разные типы

    C++ — перегрузка функции add 
    #include <iostream>

int add(int x, int y)
{
    std::cout << "add(int, int)\n";
    return x + y;
}

double add(double x, double y)
{
    std::cout << "add(double, double)\n";
    return x + y;
}

int add(int x, int y, int z) // перегрузка по количеству параметров
{
    std::cout << "add(int, int, int)\n";
    return x + y + z;
}

int main()
{
    std::cout << add(1, 2)       << '\n'; // add(int, int) → 3
    std::cout << add(1.5, 2.5)   << '\n'; // add(double, double) → 4
    std::cout << add(1, 2, 3)    << '\n'; // add(int, int, int) → 6
    return 0;
}

    Разрешение перегрузки

    Когда компилятор видит вызов перегруженной функции, он выбирает подходящую версию. Этот процесс называется разрешением перегрузки (overload resolution).

    Правила дифференциации

    📋 Что учитывается при перегрузке
    • ✅ Количество параметров
    • ✅ Типы параметров
    • ❌ Тип возвращаемого значения — не учитывается
    C++ — что работает, что нет 
    void print(int x)    {}  // OK
void print(double x) {}  // OK: другой тип параметра
void print(int x, int y) {} // OK: другое количество параметров

// ОШИБКА: отличается только тип возврата — не допускается
// int  getValue() {}
// double getValue() {} // compile error!

    Живое применение: универсальный вывод

    C++ — перегрузка print для разных типов 
    #include <iostream>
#include <string>

void print(int x)               { std::cout << "int: "    << x << '\n'; }
void print(double x)            { std::cout << "double: " << x << '\n'; }
void print(const std::string& s){ std::cout << "string: " << s << '\n'; }
void print(bool b)              { std::cout << "bool: "   << std::boolalpha << b << '\n'; }

int main()
{
    print(42);           // int: 42
    print(3.14);         // double: 3.14
    print("hello");      // string: hello
    print(true);         // bool: true
    return 0;
}
    Лучшая практика

    Используйте перегрузку функций, чтобы упростить программу — меньше имён, которые нужно помнить.
