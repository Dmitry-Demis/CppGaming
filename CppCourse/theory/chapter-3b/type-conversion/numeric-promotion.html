    Преобразования типов

    Числовые повышения

    Числовое повышение (numeric promotion) — это безопасное преобразование узкого числового типа в более широкий, который процессор обрабатывает эффективнее. Все повышения сохраняют значение — потери данных нет.

    Зачем это нужно

    32-битный процессор работает с 32-битными значениями нативно. Операции с 8-битным char или 16-битным short могут быть медленнее или требовать дополнительных инструкций. Повышения решают эту проблему и заодно устраняют дублирование функций:

    C++ — одна функция для многих типов 
    #include <iostream>

void printInt(int x) { std::cout << x << '\n'; }

int main()
{
    printInt(2);       // int — без преобразования
    short s { 3 };
    printInt(s);       // short → int (повышение)
    printInt('a');     // char → int (повышение), выведет 97
    printInt(true);    // bool → int (повышение), выведет 1
    return 0;
}

    Повышение вещественных: float → double

    Единственное правило для вещественных: float повышается до double.

    C++ — float → double 
    void printDouble(double d) { std::cout << d << '\n'; }

printDouble(5.0);   // double — без преобразования
printDouble(4.0f);  // float → double (повышение)

    💡 Важно

    Не все расширяющие преобразования являются повышениями. Например, char → short или int → long — это числовые преобразования, а не повышения. Разница важна при разрешении перегрузки функций.
