    Итоговая таблица: область видимости, время жизни, компоновка

    Три ключевых свойства переменной в C++ — область видимости, время жизни и компоновка — независимы друг от друга. Эта страница собирает всё в одном месте.

    Область видимости

    📋 Виды области видимости
    • Блочная (локальная) — от точки объявления до конца блока. Локальные переменные, параметры функций, типы объявленные внутри блока.
    • Глобальная — от точки объявления до конца файла. Глобальные переменные, функции, типы в пространстве имён.

    Время жизни

    📋 Виды времени жизни
    • Автоматическое — создаётся при объявлении, уничтожается при выходе из блока. Локальные переменные, параметры.
    • Статическое — создаётся при запуске программы, уничтожается при завершении. Глобальные переменные, статические локальные переменные.
    • Динамическое — управляется программистом через new/delete.

    Компоновка

    📋 Виды компоновки
    • Нет компоновки — каждое объявление — отдельный объект. Локальные переменные.
    • Внутренняя — одно объявление в пределах одной единицы трансляции. static глобальные, const/constexpr глобальные, безымянные пространства имён.
    • Внешняя — одно объявление во всей программе. Неконстантные глобальные, функции, extern const, inline const.

    Полная таблица переменных

    Тип Пример Область Время жизни Компоновка
    Локальная int x; Блочная Автоматическое Нет
    Статическая локальная static int s_x; Блочная Статическое Нет
    Параметр функции void f(int x) Блочная Автоматическое Нет
    Внутренняя глобальная static int g_x; Глобальная Статическое Внутренняя
    Внешняя глобальная int g_x; Глобальная Статическое Внешняя
    Inline глобальная (C++17) inline int g_x; Глобальная Статическое Внешняя
    Внутренняя const глобальная constexpr int g_x { 1 }; Глобальная Статическое Внутренняя
    Внешняя const глобальная extern const int g_x { 1 }; Глобальная Статическое Внешняя
    Inline const глобальная (C++17) inline constexpr int g_x { 1 }; Глобальная Статическое Внешняя

    Спецификаторы класса хранения

    static, extern, mutable, thread_local — это спецификаторы класса хранения. Они одновременно задают время жизни и компоновку.

    C++ — предварительные объявления 
    // Предварительные объявления:
void foo(int x);          // функция
extern int g_y;           // неконстантная глобальная
extern const int g_y;     // const глобальная
