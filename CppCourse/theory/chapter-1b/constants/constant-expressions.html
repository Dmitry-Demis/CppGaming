    Константы и constexpr

    Константные выражения и compile-time вычисления

    Компилятор C++ умеет вычислять некоторые выражения прямо во время компиляции, не оставляя работу на время выполнения программы. Это делает программы быстрее и надёжнее.

    Константные выражения

    Константное выражение — выражение, которое гарантированно вычисляется при компиляции. Оно может содержать только литералы, constexpr-переменные и операторы с такими операндами.

    C++ — что является константным выражением
    // Константные выражения:
constexpr int a { 5 };           // OK: constexpr
constexpr int b { a };           // OK: a — constexpr
constexpr long c { a + 2 };      // OK: оба операнда — constexpr
const int d { 5 };               // OK: const int с литеральным инициализатором

// НЕ константные выражения:
int e { 5 };                     // e — не const
const int f { e };               // f — const, но инициализатор не constexpr
const double g { 1.2 };          // g — const double (не int!), не constexpr

// Использование в контекстах, требующих константного выражения:
int arr[a];                      // OK: a — constexpr
// int arr2[e];                  // ОШИБКА: e — не constexpr
