    Системы счисления

    В повседневной жизни мы используем десятичную систему (основание 10). В программировании часто встречаются ещё три: двоичная, восьмеричная и шестнадцатеричная. C++ поддерживает литералы для всех четырёх.

    Таблица систем счисления

    Литералы в C++

    C++ — литералы разных систем 
    #include <iostream>

int main()
{
    int dec { 255 };      // десятичный (по умолчанию)
    int hex { 0xFF };     // шестнадцатеричный: префикс 0x
    int oct { 0377 };     // восьмеричный: префикс 0 (ноль!)
    int bin { 0b11111111 }; // двоичный: префикс 0b (C++14)

    // Все четыре — одно и то же число 255
    std::cout << dec << ' ' << hex << ' ' << oct << ' ' << bin << '\n';
    // 255 255 255 255

    // Разделитель цифр ' (C++14) — только для читаемости
    int big { 1'000'000 };
    int bits { 0b1011'0010 };

    return 0;
}
    Ловушка: ведущий ноль

    int x { 012 } — это восьмеричное число 12, равное десятичному 10, а не 12! Никогда не пиши ведущий ноль перед числом, если не хочешь восьмеричный литерал.

    Вывод в разных системах

    C++ — вывод 
    #include <bitset>
#include <iostream>

int main()
{
    int x { 255 };

    std::cout << x << '\n';              // 255 (десятичный, по умолчанию)
    std::cout << std::hex << x << '\n'; // ff (шестнадцатеричный)
    std::cout << std::oct << x << '\n'; // 377 (восьмеричный)
    std::cout << std::dec << x << '\n'; // 255 (вернуть десятичный)

    // Двоичный вывод через std::bitset
    std::cout << std::bitset<8>{ 255 } << '\n'; // 11111111

    return 0;
}

    Hex как компактная запись двоичного

    Каждая шестнадцатеричная цифра точно соответствует 4 битам. Это делает hex удобным для работы с битовыми масками и адресами памяти.
