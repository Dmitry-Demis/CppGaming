    Возвращаемые значения функций

    Функции не только выполняют действия — они могут вычислять результат и возвращать его вызывающему коду. Это делает функции по-настоящему мощным инструментом.

    Оператор return

    Чтобы вернуть значение, используется оператор return. Тип возвращаемого значения указывается перед именем функции.

    C++ — простая функция с возвратом 
    // загрузка...

    return завершает функцию

    Как только выполняется return, функция немедленно завершается. Код после return не выполняется — это удобно для раннего выхода.

    C++ — ранний выход 
    // загрузка...

    Типы возвращаемых значений

    Функция может возвращать любой тип C++: числа, строки, булевы значения и т.д.

    C++ — разные типы возврата 
    // загрузка...

    Trailing return type C++11

    В C++11 появился альтернативный синтаксис — trailing return type (тип возврата после стрелки ->). Особенно полезен с шаблонами и auto.

    C++11 — trailing return type 
    // загрузка...

    [[nodiscard]] C++17

    Атрибут [[nodiscard]] заставляет компилятор выдавать предупреждение, если возвращаемое значение функции игнорируется. Полезно для кодов ошибок и важных результатов. В C++20 можно добавить строку-пояснение: [[nodiscard("причина")]].

    C++17 — [[nodiscard]] 
    // загрузка...

    Забытый return — UB

    Отсутствие return в не-void функции — UB

    Если функция должна вернуть значение, но return не достигается — это неопределённое поведение. Включайте -Wall -Wextra чтобы компилятор предупреждал об этом.
