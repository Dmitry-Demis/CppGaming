    Введение в функции

    Программа на C++ — это набор инструкций, которые выполняются последовательно. Но по мере роста программы держать всё в main() становится неудобно. Функции позволяют разбить код на именованные блоки, которые можно вызывать многократно.

    Что такое функция

    Функция — это именованная последовательность инструкций, которая выполняет определённую задачу. Функцию можно вызвать из любого места программы, и она выполнится, после чего управление вернётся обратно.

    Зачем нужны функции
    • Повторное использование — написал один раз, вызываешь сколько угодно
    • Читаемость — код разбит на логические блоки с понятными именами
    • Тестируемость — каждую функцию можно проверить отдельно
    • Абстракция — скрываешь детали реализации за интерфейсом

    Синтаксис функции

    Функция состоит из заголовка и тела. Базовый вызов и повторное использование:

    C++ — первая функция 
    // загрузка...

    Вложенные вызовы

    Функции могут вызывать другие функции. Это основа построения сложных программ из простых блоков.

    C++ — вложенные вызовы 
    // загрузка...

    Определение и объявление

    В C++ важно различать два понятия:

    • Определение (definition) — полный код функции с телом. Компилятор генерирует машинный код.
    • Объявление (declaration / forward declaration) — только заголовок без тела, заканчивается ;. Сообщает компилятору, что функция существует.
    C++ — forward declaration 
    // загрузка...
    Порядок имеет значение

    Компилятор читает файл сверху вниз. Если вызвать функцию до её определения и без объявления — ошибка компиляции. Решения:

    • Поместить определение функции выше места вызова
    • Добавить forward declaration перед main()

    main() — тоже функция

    Правила для main()
    • Должна быть ровно одна функция main() в программе
    • Возвращает int — код завершения для ОС
    • return 0; можно опустить — компилятор добавит автоматически (C++11)
    • Нельзя вызвать main() из другой функции (это UB)
