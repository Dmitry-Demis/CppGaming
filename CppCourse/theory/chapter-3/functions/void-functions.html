    Void-функции

    Не каждая функция должна что-то вычислять. Иногда нужно просто сделать что-то — вывести текст, изменить состояние, записать в файл. Для таких функций используется тип возврата void.

    Что такое void

    void — специальный тип, означающий «ничего». Функция с типом возврата void не возвращает никакого значения.

    // загрузка...

    Ранний выход и guard clauses

    В void-функции можно использовать return; (без значения) для досрочного выхода. Проверять предусловия в начале и сразу выходить при нарушении — это паттерн guard clause. Код становится «плоским» и читаемым.

    C++ — ранний выход

    void vs возвращаемое значение

    Правило выбора
    • Используй возвращаемое значение, если функция вычисляет что-то — результат нужен вызывающему коду.
    • Используй void, если функция делает что-то — выводит, записывает, изменяет состояние.
    • Функция, которая и вычисляет, и имеет побочные эффекты — признак плохого дизайна. Старайся разделять.
