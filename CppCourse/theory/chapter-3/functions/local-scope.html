    Локальная область видимости

    Переменные, объявленные внутри функции, называются локальными. Они существуют только пока выполняется функция и недоступны снаружи.

    Локальные переменные

    Область видимости (scope) — часть программы, где имя переменной доступно. Локальная переменная видна только внутри блока { }, в котором объявлена.

    C++ — локальные переменные 
    // загрузка...

    Время жизни

    Время жизни (lifetime) — период от создания переменной до её уничтожения. Локальная переменная создаётся при входе в блок и уничтожается при выходе. Каждый вызов функции создаёт новые экземпляры локальных переменных.

    Shadowing — перекрытие имён

    Если локальная переменная имеет то же имя, что и переменная во внешней области — она её перекрывает (shadowing). Это легальный C++, но часто источник путаницы. Компиляторы предупреждают об этом с -Wshadow.

    // загрузка...
    Глобальные переменные — с осторожностью

    Глобальные переменные доступны из любой функции, что делает программу трудно предсказуемой. Предпочитай передавать данные через параметры и возвращать через return. Глобальные переменные — крайний случай.
