    Шаблоны функций

    Перегрузка решает проблему разных типов, но требует писать одинаковый код несколько раз. Шаблон функции — это рецепт, по которому компилятор сам генерирует нужную функцию для любого типа. Один шаблон вместо десяти перегрузок.

    Проблема: дублирование кода

    C++ — без шаблона: копипаст
    int    max(int x,    int y)    { return (x < y) ? y : x; }
double max(double x, double y) { return (x < y) ? y : x; }
long   max(long x,   long y)   { return (x < y) ? y : x; }
// Тело одинаковое! Только типы разные.

    Синтаксис шаблона

    C++ — шаблон функции max
    #include <iostream>

// T — параметр типа: компилятор подставит реальный тип
template <typename T>
T max(T x, T y)
{
    return (x < y) ? y : x;
}

int main()
{
    // Явное указание типа:
    std::cout << max<int>(3, 7)       << '\n'; // 7
    std::cout << max<double>(1.5, 2.5) << '\n'; // 2.5

    // Вывод типа (template argument deduction) — компилятор сам определяет T:
    std::cout << max(10, 20)     << '\n'; // T = int → 20
    std::cout << max(3.14, 2.71) << '\n'; // T = double → 3.14
    std::cout << max('a', 'z')   << '\n'; // T = char → z

    return 0;
}

    Инстанциация: как это работает

    Шаблон — это не функция. Компилятор генерирует реальную функцию при первом вызове с конкретным типом. Этот процесс называется инстанциацией.

    Живое применение: универсальный swap

    C++ — шаблонный swap
    #include <iostream>
#include <string>

template <typename T>
void mySwap(T& a, T& b)
{
    T temp { a };
    a = b;
    b = temp;
}

int main()
{
    int x { 5 }, y { 10 };
    mySwap(x, y);
    std::cout << x << ' ' << y << '\n'; // 10 5

    std::string s1 { "hello" }, s2 { "world" };
    mySwap(s1, s2);
    std::cout << s1 << ' ' << s2 << '\n'; // world hello

    double a { 1.1 }, b { 2.2 };
    mySwap(a, b);
    std::cout << a << ' ' << b << '\n'; // 2.2 1.1

    return 0;
}
    💡 Шаблон vs перегрузка

    Перегрузка — когда поведение для разных типов разное. Шаблон — когда поведение одинаковое, только типы разные. Шаблоны можно тоже перегружать.
