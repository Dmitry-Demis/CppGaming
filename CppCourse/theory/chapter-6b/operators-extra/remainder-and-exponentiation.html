    Остаток и возведение в степень

    Оператор остатка (%)

    Оператор % возвращает остаток от целочисленного деления. Например, 7 % 4 = 3, потому что 7 / 4 = 1 с остатком 3. Оператор работает только с целыми операндами.

    Чаще всего он используется для проверки делимости: если x % y == 0, то x делится на y без остатка.

    C++ — оператор остатка 
    // загрузка...

    Остаток с отрицательными числами

    Результат x % y всегда имеет знак первого операнда (x). Например: -6 % 4 = -2, а 6 % -4 = 2.

    Лучшая практика

    Сравнивайте результат % с 0, а не с 1. Функция isOdd(x) через x % 2 == 1 сломается на отрицательных числах — используйте x % 2 != 0.

    Переполнение

    Целочисленное возведение в степень почти всегда переполняет тип. Именно поэтому такой функции нет в стандартной библиотеке. Используйте std::int64_t и проверяйте переполнение.
