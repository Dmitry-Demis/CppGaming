    Ключевые слова и именование

    Хорошие имена — половина читаемого кода. Понимание ключевых слов и правил именования поможет писать понятный C++.

    Ключевые слова

    Ключевые слова (keywords) — зарезервированные слова языка. Их нельзя использовать как имена переменных или функций. В C++ их около 90.

    Часто встречающиеся ключевые слова

    int   double   char   bool   void   return   if   else   for   while   do   break   continue   class   struct   namespace   const   auto   true   false

    Правила именования идентификаторов

    C++ — именование идентификаторов 
    // загрузка...
    Правила (обязательные)
    • Только буквы, цифры и подчёркивание _
    • Не может начинаться с цифры
    • Чувствительно к регистру: value ? Value
    • Нельзя использовать ключевые слова
    • Нельзя начинать с _ (зарезервировано для компилятора)

    Соглашения об именовании

    ? Рекомендации
    • Переменные и функции: camelCase или snake_case
    • Константы: UPPER_CASE или kConstantName
    • Классы и типы: PascalCase
    • Имя должно описывать назначение: playerScore, не ps
    • Избегайте однобуквенных имён (кроме счётчиков i, j)
