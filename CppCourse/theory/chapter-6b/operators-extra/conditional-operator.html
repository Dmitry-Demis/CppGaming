    Тернарный оператор

    Синтаксис
    условие ? выражение1 : выражение2

    Оператор ?: — единственный тернарный оператор C++. Если условие истинно, вычисляется и возвращается выражение1, иначе — выражение2.

    Сравнение с if-else

    Конструкция if-else — это инструкция, а ?: — выражение. Это позволяет использовать тернарный оператор там, где инструкция недопустима, например при инициализации constexpr-переменной:

    C++ — тернарный оператор 
    // загрузка...

    Скобки

    Тернарный оператор имеет низкий приоритет. Оборачивайте всё выражение в скобки при использовании в составных выражениях, а условие — если оно содержит операторы:

    C++ — скобки в тернарном операторе 
    // загрузка...

    Совместимость типов

    Типы второго и третьего операндов должны совпадать или быть приводимы друг к другу. Смешивание знаковых и беззнаковых типов может дать неожиданный результат.

    Когда использовать

    Тернарный оператор хорош для инициализации переменной одним из двух значений, передачи аргумента или возврата из функции. Избегайте его в сложных вложенных выражениях.
