    Операторы сравнения

    Операторы сравнения позволяют сравнивать два значения и возвращают булево значение true или false.

    Шесть операторов сравнения
    Оператор Символ Значение
    Больше>true, если x > y
    Меньше<true, если x < y
    Больше или равно>=true, если x >= y
    Меньше или равно<=true, если x <= y
    Равно==true, если x == y
    Не равно!=true, если x != y
    Лучшая практика

    Не добавляйте лишние == true или == false в условия. Пишите if (b) вместо if (b == true), и if (!b) вместо if (b == false).

    Сравнение вещественных чисел

    Вещественные числа хранятся с погрешностью округления, поэтому прямое сравнение через == и != ненадёжно. Например, 0.1 + 0.1 + ... (10 раз) != 1.0.

    Предупреждение

    Избегайте == и != для вещественных чисел, если они были вычислены. Операторы <, >, <=, >= надёжны, когда значения существенно различаются.
