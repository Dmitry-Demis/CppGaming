    Перечисления (unscoped enum)

    Перечисление (enumeration, enum) — это составной тип данных, значения которого ограничены набором именованных символических констант, называемых перечислителями. Перечисления позволяют заменить «магические числа» читаемыми именами.

    Зачем нужны перечисления

    Без перечислений цвет яблока пришлось бы хранить как число:

    C++ — без enum (плохо)
    int appleColor{ 0 }; // 0 = red? green? кто знает...
int shirtColor{ 1 };

    С перечислением код становится самодокументируемым:

    C++ — с enum (хорошо)
    enum Color
{
    red,
    green,
    blue,
};

int main()
{
    Color apple { red };   // очевидно: яблоко красное
    Color shirt { green }; // рубашка зелёная

    // Color socks { white }; // ошибка: white не является перечислителем Color
    // Color hat { 2 };       // ошибка: 2 не является перечислителем Color

    return 0;
}

    Синтаксис

    Перечисление объявляется ключевым словом enum, за которым следует имя типа и список перечислителей в фигурных скобках. Перечислители разделяются запятыми. Определение заканчивается точкой с запятой.

    📋 Правила именования
    • Имя перечисления начинается с заглавной буквы: Color, Direction
    • Имена перечислителей начинаются со строчной буквы: red, north
    • Перечислители являются неявно constexpr
