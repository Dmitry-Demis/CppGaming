    Перечисления (unscoped enum)

    Перечисление (enumeration) — это составной тип данных, значения которого ограничены набором именованных символических констант, называемых перечислителями. Перечисления делают код самодокументируемым и исключают «магические числа».

    Определение перечисления

    Unscoped-перечисление объявляется ключевым словом enum:

    C++ — объявление enum
    enum Color
{
    red,    // 0
    green,  // 1
    blue,   // 2 — trailing comma рекомендуется
};

int main()
{
    Color apple { red };   // OK
    Color shirt { green }; // OK

    // Color socks { white }; // ошибка: white не является перечислителем Color
    // Color hat { 2 };       // ошибка: 2 не является перечислителем Color
    return 0;
}
    📋 Терминология
    • Перечисление / enumerated type — сам тип (например, Color)
    • Перечислитель / enumerator — конкретное именованное значение (например, red)
    • Перечислители неявно являются constexpr

    Именование

    Лучшая практика

    Имя перечисления начинается с заглавной буквы (Color), имена перечислителей — со строчной (red, green).

    Область видимости

    Unscoped-перечисления помещают перечислители в ту же область видимости, где определено само перечисление. Это может приводить к конфликтам имён:

    C++ — конфликт имён
    enum Color  { red, green, blue };
enum Feeling{ happy, tired, blue }; // ошибка: blue уже определён

    Решение — поместить перечисление в пространство имён:

    C++ — enum в namespace
    namespace Color   { enum Color   { red, green, blue }; }
namespace Feeling { enum Feeling { happy, tired, blue }; }

int main()
{
    Color::Color   paint { Color::blue };
    Feeling::Feeling me  { Feeling::blue }; // нет конфликта
    return 0;
}

    Применение

    Перечисления удобны для небольших наборов связанных констант: дни недели, масти карт, коды ошибок, типы предметов в игре.

    C++ — коды ошибок через enum
    enum FileReadResult
{
    readResultSuccess,
    readResultErrorFileOpen,
    readResultErrorFileRead,
    readResultErrorFileParse,
};

FileReadResult readFile()
{
    if (!openFile())  return readResultErrorFileOpen;
    if (!readFile())  return readResultErrorFileRead;
    if (!parseFile()) return readResultErrorFileParse;
    return readResultSuccess;
}

// Вызывающий код читается как текст:
if (readFile() == readResultSuccess) { /* ... */ }

    Сравнение перечислителей

    C++
    #include <iostream>
enum Color { red, green, blue };

int main()
{
    Color shirt { blue };
    if (shirt == blue)
        std::cout << "Рубашка синяя\n";
    return 0;
}
