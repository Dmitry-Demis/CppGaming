    Глобальные переменные

    Переменные, объявленные вне любой функции, называются глобальными. В отличие от локальных, они доступны из любой точки файла после объявления.

    Объявление глобальных переменных

    По соглашению глобальные переменные объявляют в начале файла, после директив #include:

    C++ — глобальная переменная 
    #include <iostream>

int g_x {}; // глобальная переменная g_x

void doSomething()
{
    g_x = 3;
    std::cout << g_x << '\n';
}

int main()
{
    doSomething();
    std::cout << g_x << '\n'; // 3

    g_x = 5;
    std::cout << g_x << '\n'; // 5

    return 0;
} // g_x выходит из области видимости здесь

    Область видимости: глобальная

    Глобальные переменные имеют глобальную область видимости — они видны от точки объявления до конца файла. Переменные внутри пространства имён тоже являются глобальными:

    C++ — глобальная переменная в namespace 
    namespace Foo
{
    int g_x {}; // глобальная переменная внутри пространства имён
}

int main()
{
    Foo::g_x = 5; // доступна через квалифицированное имя
    return 0;
}
    💡 Ключевой момент

    Переменные внутри пространства имён — тоже глобальные переменные.

    Время жизни: статическое

    Глобальные переменные создаются при запуске программы (до main()) и уничтожаются при её завершении. Это называется статическим временем жизни. Переменные со статическим временем жизни иногда называют статическими переменными.

    Соглашение об именовании

    Принято добавлять префикс g_ к именам глобальных переменных. Это помогает сразу видеть, что переменная глобальная, и избегать случайного перекрытия имён:

    C++ — соглашение g_ 
    int g_userCount {};      // глобальная, сразу видно
int g_maxConnections {}; // глобальная, сразу видно

    Инициализация глобальных переменных

    В отличие от локальных, глобальные переменные нулевой инициализируются по умолчанию. Но лучше всегда инициализировать явно:

    C++ — варианты инициализации 
    int g_x;        // нулевая инициализация по умолчанию (0)
int g_y {};     // явная нулевая инициализация
int g_z { 1 }; // инициализация значением

const int g_c { 42 };       // константная глобальная
constexpr int g_ce { 100 }; // constexpr глобальная
    Лучшая практика

    Предпочитайте объявлять глобальные переменные внутри пространства имён, а не в глобальном пространстве.
