    Блоки и составные операторы

    Блок (или составной оператор) — это группа из нуля и более операторов, заключённых в фигурные скобки { }. Компилятор воспринимает весь блок как единый оператор. Блоки можно использовать везде, где допустим одиночный оператор. Точка с запятой после закрывающей скобки не нужна.

    Что такое блок

    Тело функции — это блок. Именно поэтому вы уже видели блоки с первого урока:

    C++ — тело функции как блок 
    int add(int x, int y)
{ // начало блока
    return x + y;
} // конец блока (без точки с запятой)

int main()
{ // начало блока
    int value {};
    add(3, 4);
    return 0;
} // конец блока

    Вложенные блоки

    Блоки можно вкладывать друг в друга. Внешний блок называют внешним, внутренний — вложенным. Функции вкладывать нельзя, а блоки — можно.

    C++ — вложенные блоки 
    int main()
{ // внешний блок

    int value {};

    { // вложенный блок
        add(3, 4);
    } // конец вложенного блока

    return 0;
} // конец внешнего блока

    Блоки и условные операторы

    Самое частое применение блоков — выполнение нескольких операторов внутри if/else:

    C++ — блок внутри if/else 
    #include <iostream>

int main()
{
    int value {};
    std::cin >> value;

    if (value >= 0)
    { // блок для true-ветки
        std::cout << value << " — положительное число\n";
        std::cout << "Удвоенное: " << value * 2 << '\n';
    }
    else
    { // блок для false-ветки
        std::cout << value << " — отрицательное число\n";
        std::cout << "Модуль: " << -value << '\n';
    }

    return 0;
}

    Уровень вложенности

    Уровень вложенности — максимальная глубина блоков, в которых вы можете одновременно находиться в данной функции. Стандарт C++ требует поддержки не менее 256 уровней, но на практике уже 3–4 уровня делают код трудночитаемым.

    C++ — три уровня вложенности 
    #include <iostream>

int main()
{ // уровень 1
    int value {};
    std::cin >> value;

    if (value > 0)
    { // уровень 2
        if ((value % 2) == 0)
        { // уровень 3
            std::cout << value << " — положительное чётное\n";
        }
        else
        { // уровень 3
            std::cout << value << " — положительное нечётное\n";
        }
    }

    return 0;
}
    💡 Правило трёх уровней

    Старайтесь не превышать 3 уровня вложенности. Если функция требует большей глубины — это сигнал вынести часть логики в отдельную функцию.

    Лучшая практика

    Держите уровень вложенности не более 3. Глубоко вложенный код — кандидат на рефакторинг.
