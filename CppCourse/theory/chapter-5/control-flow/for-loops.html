    Цикл for

    for — наиболее часто используемый цикл в C++. Он объединяет инициализацию, условие и итерацию в одной строке, что делает код компактным и читаемым.

    Три части заголовка

    Синтаксис
    for (инициализация; условие; итерация)
    инструкция;

    Порядок выполнения:

    1. Инициализация — выполняется один раз перед первой итерацией.
    2. Условие — проверяется перед каждой итерацией. Если ложно — цикл завершается.
    3. Тело — выполняется при истинном условии.
    4. Итерация — выполняется после каждого тела, затем снова шаг 2.

    Все три части необязательны. for (;;) — бесконечный цикл, эквивалентный while (true).

    Range-based for (for-each)

    Для перебора всех элементов последовательности используйте range-based for:

    Синтаксис range-based for
    for (тип элемент : последовательность)
    инструкция;

    Он работает с массивами, std::array, std::vector и любым типом, предоставляющим итераторы. Для крупных объектов используйте const auto& вместо копирования.

    Вложенные циклы

    Циклы for можно вкладывать друг в друга. Для каждой итерации внешнего цикла внутренний цикл выполняется полностью. Классический пример — таблица умножения.

    Производительность вложенных циклов

    При n уровнях вложенности и m итерациях на каждом уровне тело выполняется mn раз. Глубокая вложенность быстро становится дорогой — следите за сложностью.
