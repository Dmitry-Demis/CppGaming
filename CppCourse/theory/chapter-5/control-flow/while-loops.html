    Цикл while

    Цикл — конструкция, которая повторяет блок кода до тех пор, пока выполняется условие. while — наиболее базовая форма цикла в C++.

    Синтаксис и итерации

    Синтаксис
    while (условие)
    инструкция;

// или с блоком:
while (условие)
{
    // тело цикла
}

    Каждое выполнение тела цикла называется итерацией. Перед каждой итерацией вычисляется условие. Если оно истинно — тело выполняется. Если ложно — цикл завершается.

    C++ — while: базовые примеры 
    // загрузка...

    Условие ложно с самого начала

    Если условие ложно при первой проверке, тело цикла не выполняется ни разу. Выполнение продолжается с инструкции после цикла. Это нормальное поведение — while не гарантирует хотя бы одно выполнение.

    Намеренный бесконечный цикл

    while (true) создаёт цикл, который не завершится сам по себе. Выход из него — через break, return или исключение. Такой паттерн используется в игровых циклах, серверах и обработчиках событий.

    Паттерн бесконечного цикла
    while (true)
{
    // обработка события
    if (shouldExit())
        break;
}

    Опасность беззнаковых счётчиков

    Переполнение беззнакового счётчика

    Беззнаковые целые (unsigned int, size_t) не могут быть отрицательными. При уменьшении ниже нуля происходит переполнение: значение «оборачивается» до максимума типа. 

    unsigned int u {3};
while (u >= 0)  // условие ВСЕГДА истинно — бесконечный цикл!
{
    --u;  // при u==0 следующее значение: 4294967295
}

    Используйте знаковые типы (int) для счётчиков с уменьшением.
