    break и continue

    break и continue позволяют управлять ходом цикла без введения дополнительных флаговых переменных.

    Оператор break

    break немедленно завершает ближайший цикл (for, while, do-while) или блок switch. Выполнение продолжается с первой инструкции после закрывающей скобки.

    C++ — break и continue 
    // загрузка...

    break против return

    Ключевое различие
    break return
    Завершает ближайший цикл или switch Завершает всю функцию
    Выполнение продолжается после цикла Выполнение возвращается вызывающему коду

    Оператор continue

    continue пропускает оставшуюся часть текущей итерации и переходит к следующей. В for выполняется часть итерации заголовка, в while/do-while — сразу проверяется условие.

    Опасность continue в while

    Если инкремент счётчика стоит после continue, он никогда не выполнится при срабатывании continue. Счётчик застревает — цикл становится бесконечным. 

    int i {0};
while (i < 5)
{
    if (i == 2)
        continue;  // БЕСКОНЕЧНЫЙ ЦИКЛ при i==2
    ++i;           // никогда не выполнится при i==2
}

    Решение: поместить инкремент до continue.
