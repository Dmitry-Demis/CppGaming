    Продвинутые constexpr

    constexpr-функции

    Обычные функции нельзя вызывать в константных выражениях. constexpr-функция — это функция, которую разрешено использовать в константных выражениях и вычислять на этапе компиляции.

    Проблема

    C++ — ошибка компиляции
    #include <iostream>

double calcCircumference(double radius)
{
    constexpr double pi { 3.14159265359 };
    return 2.0 * pi * radius;
}

int main()
{
    constexpr double c { calcCircumference(3.0) }; // ошибка: не constexpr
    std::cout << c << '\n';
    return 0;
}

    Решение: constexpr-функция

    C++
    #include <iostream>

constexpr double calcCircumference(double radius) // теперь constexpr
{
    constexpr double pi { 3.14159265359 };
    return 2.0 * pi * radius;
}

int main()
{
    constexpr double c { calcCircumference(3.0) }; // OK: вычисляется при компиляции
    std::cout << c << '\n'; // 18.8496
    return 0;
}

    Когда вычисляется при компиляции?

    📋 Правила вычисления
    • Гарантировано при компиляции — если результат используется там, где требуется константное выражение (например, инициализатор constexpr-переменной)
    • Возможно при компиляции — если все аргументы — константные выражения, но результат не обязан быть константным
    • При выполнении — если хотя бы один аргумент не является константным выражением
    C++ — четыре случая
    constexpr int greater(int x, int y) { return x > y ? x : y; }

int main()
{
    constexpr int g { greater(5, 6) };  // 1: всегда при компиляции
    std::cout << greater(5, 6);         // 2: возможно при компиляции
    int x { 5 };
    std::cout << greater(x, 6);         // 3: скорее всего при выполнении
    std::cin >> x;
    std::cout << greater(x, 6);         // 4: всегда при выполнении
    return 0;
}

    constexpr-функции неявно inline

    Чтобы компилятор мог вычислить функцию при компиляции, ему нужно видеть её полное определение. Поэтому constexpr-функции неявно inline и обычно определяются в заголовочных файлах.

    Лучшая практика

    Если функция может быть частью константного выражения — сделайте её constexpr. Всегда проверяйте её в контексте, требующем константного выражения.
