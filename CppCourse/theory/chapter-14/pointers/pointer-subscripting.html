    Указатели

    Арифметика указателей и индексация

    Указатели — не просто адреса. К ним можно применять арифметические операции, что позволяет перемещаться по памяти и обходить массивы без явных индексов.

    Арифметика указателей

    ptr + n возвращает адрес объекта, который находится на n позиций вперёд от ptr. Шаг зависит от типа: для int* это 4 байта, для double* — 8 байт.

    C++ — арифметика указателей 
    #include <iostream>

int main()
{
    int x {};
    const int* ptr { &x }; // предположим, адрес 0x1000

    std::cout << ptr     << '\n'; // 0x1000
    std::cout << ptr + 1 << '\n'; // 0x1004 (следующий int)
    std::cout << ptr + 2 << '\n'; // 0x1008

    std::cout << ptr - 1 << '\n'; // 0x0FFC (предыдущий int)

    return 0;
}

    Индексация через указатель

    Оператор ptr[n] — это синтаксический сахар для *(ptr + n). Именно поэтому массивы в C++ индексируются с нуля: arr[0] = *(arr + 0) = *arr.

    C++ — индексация = арифметика + разыменование 
    #include <iostream>

int main()
{
    const int arr[] { 9, 7, 5, 3, 1 };
    const int* ptr { arr }; // ptr указывает на arr[0]

    // Эти выражения эквивалентны:
    std::cout << arr[2]      << '\n'; // 5
    std::cout << *(arr + 2)  << '\n'; // 5
    std::cout << ptr[2]      << '\n'; // 5
    std::cout << *(ptr + 2)  << '\n'; // 5

    // Указатель на середину массива:
    const int* mid { &arr[2] }; // указывает на arr[2]
    std::cout << mid[0]  << '\n'; // 5 (arr[2])
    std::cout << mid[1]  << '\n'; // 3 (arr[3])
    std::cout << mid[-1] << '\n'; // 7 (arr[1]) — отрицательный индекс!

    return 0;
}

    Обход массива через указатель

    Классический паттерн: begin и end указатели. Именно так работают range-based for циклы под капотом.

    C++ — обход через begin/end 
    #include <iostream>
#include <iterator> // std::size

int main()
{
    constexpr int arr[] { 9, 7, 5, 3, 1 };

    const int* begin { arr };                    // указывает на arr[0]
    const int* end   { arr + std::size(arr) };   // указывает на один-за-последним

    for (; begin != end; ++begin)
        std::cout << *begin << ' '; // 9 7 5 3 1
    std::cout << '\n';

    // Это эквивалентно range-for:
    for (int x : arr)
        std::cout << x << ' ';

    return 0;
}
    Почему индексы начинаются с 0

    Потому что arr[n] = *(arr + n). Первый элемент — это *(arr + 0) = *arr. Нулевой индекс делает математику проще и эффективнее.
