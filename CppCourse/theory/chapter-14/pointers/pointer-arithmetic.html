    Указатели

    Арифметика указателей и массивы

    Указатели поддерживают арифметические операции. Прибавление числа к указателю перемещает его на несколько объектов вперёд, а не на байты. Это тесно связано с тем, как работают массивы.

    Арифметика указателей

    C++ — инкремент и сложение
    #include <iostream>

int main()
{
    int n { 10 };
    int* p { &n };

    std::cout << p  << '\n'; // адрес, например 0x1000
    p++;                      // адрес увеличивается на sizeof(int) = 4
    std::cout << p  << '\n'; // 0x1004

    double d { 3.14 };
    double* pd { &d };
    pd += 2; // адрес увеличивается на 2 * sizeof(double) = 16
    // pd теперь указывает на 2 объекта double вперёд

    return 0;
}

    Указатели и массивы

    Имя массива — это адрес его первого элемента. Индексация arr[i] эквивалентна *(arr + i).

    C++ — обход массива через указатель
    #include <iostream>

int main()
{
    int nums[] { 1, 2, 3, 4, 5 };

    // Через индекс:
    for (int i = 0; i < 5; ++i)
        std::cout << nums[i] << ' ';
    std::cout << '\n';

    // Через указатель (эквивалентно):
    for (int* p = nums; p < nums + 5; ++p)
        std::cout << *p << ' ';
    std::cout << '\n';

    // nums[2] == *(nums + 2)
    std::cout << nums[2] << ' ' << *(nums + 2) << '\n'; // 3 3

    return 0;
}
    Лучшая практика

    В современном C++ предпочитайте range-for или итераторы вместо ручной арифметики указателей. Арифметика указателей полезна при работе с C API и низкоуровневым кодом.
