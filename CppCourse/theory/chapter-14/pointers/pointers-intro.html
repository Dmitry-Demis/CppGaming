    Указатели

    Введение в указатели

    Указатель — это переменная, которая хранит адрес памяти другого объекта. Указатели дают прямой доступ к памяти и являются одним из самых мощных (и опасных) инструментов C++.

    Оператор взятия адреса &

    C++ — адрес переменной
    #include <iostream>

int main()
{
    int x { 42 };
    std::cout << x  << '\n';  // значение: 42
    std::cout << &x << '\n';  // адрес: 0x7ffd... (шестнадцатеричный)
    return 0;
}

    Объявление и инициализация указателя

    C++ — указатель
    #include <iostream>

int main()
{
    int x { 5 };
    int* ptr { &x }; // ptr хранит адрес x

    std::cout << ptr  << '\n'; // адрес x
    std::cout << *ptr << '\n'; // значение по адресу (разыменование): 5

    *ptr = 99; // изменяем x через указатель
    std::cout << x << '\n';    // 99

    int y { 7 };
    ptr = &y; // указатель теперь указывает на y
    std::cout << *ptr << '\n'; // 7

    return 0;
}

    Указатели vs ссылки

    Свойство Ссылка int& Указатель int*
    Инициализация Обязательна Необязательна (но нужна!)
    Перепривязка Нельзя Можно
    Может быть null Нет Да (nullptr)
    Разыменование Неявное Явное (*ptr)
    Безопасность Выше Ниже (UB при неверном использовании)
    Лучшая практика

    Всегда инициализируйте указатели. Неинициализированный указатель («дикий указатель») содержит мусорный адрес — разыменование приведёт к неопределённому поведению.
