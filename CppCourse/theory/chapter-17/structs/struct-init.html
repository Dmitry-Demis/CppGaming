    Структуры

    Инициализация структур

    Структуры являются агрегатами, поэтому для их инициализации используется агрегатная инициализация — список значений в фигурных скобках.

    Агрегатная инициализация

    C++
    struct Employee
{
    int id {};
    int age {};
    double wage {};
};

int main()
{
    Employee frank = { 1, 32, 60000.0 }; // copy-list (допустимо)
    Employee joe   { 2, 28, 45000.0 };   // list-init (предпочтительно)
    return 0;
}

    Члены инициализируются в порядке объявления. Если значений меньше, чем членов — остальные value-инициализируются (или используют default member initializer).

    Designated initializers (C++20)

    Позволяют явно указать, какое значение к какому члену относится:

    C++20
    struct Foo { int a{}; int b{}; int c{}; };

int main()
{
    Foo f1 { .a{1}, .c{3} };   // b = 0 (value-init)
    Foo f2 { .a = 1, .c = 3 }; // то же самое

    // Foo f3 { .b{2}, .a{1} }; // ошибка: порядок должен совпадать с объявлением
    return 0;
}
    Лучшая практика

    Добавляйте новые члены в конец структуры, чтобы не сдвигать существующие инициализаторы.

    Присваивание через список

    C++
    Employee joe { 1, 32, 60000.0 };
joe = { joe.id, 33, 66000.0 }; // день рождения и повышение

// C++20: designated assignment
joe = { .id = joe.id, .age = 33, .wage = 66000.0 };

    Const-структуры

    C++
    struct Rectangle { double length{}; double width{}; };

const Rectangle unit { 1.0, 1.0 };
const Rectangle zero {};  // value-init всех членов
