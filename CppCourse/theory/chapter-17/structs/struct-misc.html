    Вложенные структуры

    Структура может содержать другую структуру как член:

    C++
    #include <iostream>

struct Employee { int id{}; int age{}; double wage{}; };

struct Company
{
    int numberOfEmployees{};
    Employee CEO{};
};

int main()
{
    Company myCompany { 7, { 1, 32, 55000.0 } };
    std::cout << myCompany.CEO.wage << '\n'; // 55000
    return 0;
}

    Структуры должны владеть своими данными

    Члены структуры должны быть владельцами данных, а не наблюдателями. Используйте std::string, а не std::string_view для строковых членов:

    C++ — опасный пример
    struct Viewer { std::string_view name{}; }; // опасно!
struct Owner  { std::string name{};      }; // безопасно

std::string getName() { return "Alice"; }

int main()
{
    Owner  o { getName() }; // OK: копирует строку
    Viewer v { getName() }; // UB: string_view на уничтоженный временный объект
    return 0;
}

    Размер структуры и выравнивание

    Компилятор может добавлять padding (байты заполнения) между членами для выравнивания. Поэтому размер структуры может быть больше суммы размеров её членов:

    C++
    #include <iostream>

struct Foo1 { short a{}; int b{}; short c{}; }; // 12 байт (padding!)
struct Foo2 { int b{};   short a{}; short c{}; }; // 8 байт

int main()
{
    std::cout << sizeof(Foo1) << '\n'; // 12
    std::cout << sizeof(Foo2) << '\n'; // 8
    return 0;
}
    💡 Совет

    Объявляйте члены в порядке убывания размера, чтобы минимизировать padding.
