    Структуры

    Введение в структуры

    Структура (struct) — это пользовательский тип данных, позволяющий объединить несколько переменных в одну сущность. Вместо того чтобы хранить имя, возраст и зарплату сотрудника в трёх отдельных переменных, можно создать тип Employee, содержащий все три поля.

    Определение структуры

    C++ — определение struct
    struct Employee
{
    int id {};
    int age {};
    double wage {};
}; // обязательная точка с запятой!
    📋 Терминология
    • Переменные внутри структуры называются членами данных (data members / member variables)
    • Определение структуры не создаёт объект — только описывает тип
    • Имя структуры начинается с заглавной буквы

    Создание объектов

    C++
    Employee joe {};   // объект joe типа Employee
Employee frank {}; // объект frank типа Employee

    Доступ к членам

    Для доступа к члену используется оператор точки (.):

    C++
    #include <iostream>

struct Employee
{
    int id {};
    int age {};
    double wage {};
};

int main()
{
    Employee joe {};
    joe.id   = 14;
    joe.age  = 32;
    joe.wage = 60000.0;

    Employee frank {};
    frank.id   = 15;
    frank.age  = 28;
    frank.wage = 45000.0;

    int totalAge { joe.age + frank.age };
    std::cout << "Суммарный возраст: " << totalAge << '\n';

    if (joe.wage > frank.wage)
        std::cout << "Joe зарабатывает больше\n";
    else
        std::cout << "Frank зарабатывает не меньше\n";

    ++joe.age; // день рождения Joe
    return 0;
}
