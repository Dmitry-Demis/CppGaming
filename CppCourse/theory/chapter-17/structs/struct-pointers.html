    Указатели на структуры и оператор ->

    Ссылки и оператор точки

    Ссылка на структуру работает как сам объект — оператор . применяется напрямую:

    C++
    struct Employee { int id{}; int age{}; double wage{}; };

void print(const Employee& e)
{
    std::cout << e.id << ' ' << e.age << '\n'; // . работает через ссылку
}

    Оператор -> для указателей

    Через указатель нельзя использовать . напрямую. Оператор -> делает разыменование и выбор члена за один шаг:

    C++
    #include <iostream>

struct Employee { int id{}; int age{}; double wage{}; };

int main()
{
    Employee joe { 1, 34, 65000.0 };
    Employee* ptr { &joe };

    // (*ptr).id — работает, но некрасиво
    std::cout << ptr->id << '\n';   // эквивалентно (*ptr).id
    std::cout << ptr->age << '\n';
    return 0;
}
    Лучшая практика

    При доступе к членам через указатель всегда используйте ->, а не (*ptr)..

    Цепочки стрелок

    C++
    struct Point    { double x{}; double y{}; };
struct Triangle { Point* a{}; Point* b{}; Point* c{}; };

int main()
{
    Point a{1,2}, b{3,7}, c{10,2};
    Triangle tr { &a, &b, &c };
    Triangle* ptr { &tr };

    std::cout << ptr -> c -> y << '\n'; // 2
    return 0;
}

    Смешанный доступ

    C++
    struct Paw    { int claws{}; };
struct Animal { std::string name{}; Paw paw{}; };

int main()
{
    Animal puma { "Puma", { 5 } };
    Animal* ptr { &puma };

    // ptr — указатель → ->
    // paw — не указатель → .
    std::cout << (ptr->paw).claws << '\n'; // 5
    return 0;
}
