    Передача и возврат структур

    Передача по const-ссылке

    Структуры обычно передаются по const-ссылке, чтобы избежать копирования:

    C++
    #include <iostream>

struct Employee { int id{}; int age{}; double wage{}; };

void printEmployee(const Employee& e)
{
    std::cout << "ID: "   << e.id   << '\n';
    std::cout << "Age: "  << e.age  << '\n';
    std::cout << "Wage: " << e.wage << '\n';
}

int main()
{
    Employee joe { 14, 32, 24.15 };
    Employee frank { 15, 28, 18.27 };
    printEmployee(joe);
    printEmployee(frank);
    return 0;
}

    Временные объекты

    Если объект используется только один раз, можно передать временный объект напрямую:

    C++
    printEmployee(Employee { 14, 32, 24.15 }); // явный тип (предпочтительно)
printEmployee({ 15, 28, 18.27 });           // тип выводится из параметра

    Возврат структуры

    Функция может вернуть структуру — это удобный способ вернуть несколько значений:

    C++
    struct Point3d { double x{0.0}; double y{0.0}; double z{0.0}; };

Point3d getZeroPoint()
{
    return Point3d { 0.0, 0.0, 0.0 }; // временный объект
    // или короче:
    // return { 0.0, 0.0, 0.0 };
    // или ещё короче:
    // return {};
}

int main()
{
    Point3d zero { getZeroPoint() };
    if (zero.x == 0.0 && zero.y == 0.0 && zero.z == 0.0)
        std::cout << "Нулевая точка\n";
    return 0;
}
    Важно

    Структуры, определённые внутри функции, возвращайте по значению — не по ссылке, чтобы не получить висячую ссылку.
