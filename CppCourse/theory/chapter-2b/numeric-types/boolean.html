Логический тип bool

Тип bool — простейший из фундаментальных типов: он принимает всего два значения — true и false. Несмотря на кажущуюся тривиальность, понимание правил преобразований и поведения в выражениях предотвращает целый класс трудноуловимых ошибок.

Основы типа bool

[basic.fundamental] §6.7.1 — ISO/IEC 14882 Values of type bool are either true or false. A value of type bool participates in integral promotions. An rvalue of type bool converted to type int, with false becoming zero and true becoming one.
Значения true (1) и false (0)
Размер обычно 1 байт (стандарт не фиксирует)
true > int 1 (гарантировано)
false > int 0 (гарантировано)
C++ 
#include <iostream>

int main() {
    bool t{true};
    bool f{false};

    // По умолчанию cout выводит 1/0
    std::cout << "true  = " << t << '\n'; // 1
    std::cout << "false = " << f << '\n'; // 0

    // С boolalpha выводится "true"/"false"
    std::cout << std::boolalpha;
    std::cout << "true  = " << t << '\n'; // true
    std::cout << "false = " << f << '\n'; // false

    // bool в арифметике — интегральное повышение до int
    std::cout << std::noboolalpha;
    std::cout << "true + true = "  << (t + t) << '\n'; // 2
    std::cout << "true * 5    = "  << (t * 5) << '\n'; // 5

    return 0;
}

Преобразования в bool и из bool

Любой тип неявно преобразуется в bool по правилу: нулевое/пустое/нулевой указатель > false, всё остальное > true.

Значение> boolПример
0 (int)falsebool b = 0;
Любое ненулевое числоtruebool b = 42; > true
0.0 (double)falsebool b = 0.0;
nullptrfalsebool b = nullptr;
Ненулевой указательtruebool b = &x;
"" (пустая строка)true (!)Строковый литерал — это указатель, ? nullptr
C++ 
#include <iostream>

int main() {
    std::cout << std::boolalpha;

    // int > bool
    bool b1{0};    // false
    bool b2{1};    // true
    bool b3{-42};  // true  (ненулевое!)
    bool b4{100};  // true

    std::cout << b1 << ' ' << b2 << ' ' << b3 << ' ' << b4 << '\n';
    // false true true true

    // double > bool
    bool b5{0.0};   // false
    bool b6{0.001}; // true

    // Указатель > bool
    int x{5};
    bool b7{nullptr}; // false
    bool b8{&x};      // true

    std::cout << b5 << ' ' << b6 << ' ' << b7 << ' ' << b8 << '\n';
    // false true false true

    return 0;
}

Логические операторы

Результат операторов сравнения и логических операторов всегда имеет тип bool:

ОператорОписаниеПримерРезультат
&&Логическое Иtrue && falsefalse
||Логическое ИЛИtrue || falsetrue
!Логическое НЕ!truefalse
==Равно3 == 3true
!=Не равно3 != 4true
<, >, <=, >=Сравнение3 < 5true
C++ 
#include <iostream>

int main() {
    int a{5}, b{10};
    std::cout << std::boolalpha;

    // Операторы сравнения возвращают bool
    std::cout << (a < b)  << '\n'; // true
    std::cout << (a == b) << '\n'; // false
    std::cout << (a != b) << '\n'; // true

    // Логические операторы — ленивые вычисления (short-circuit)
    bool flag{(a > 0) && (b < 100)}; // оба выполняются
    bool lazy{(a < 0) && (b < 100)}; // правая часть НЕ вычисляется!

    std::cout << flag << '\n'; // true
    std::cout << lazy << '\n'; // false (b<100 не вычислялось)

    // Практический пример: защита от деления на ноль
    int divisor{0};
    if (divisor != 0 && a / divisor > 2) { /* безопасно */ }

    return 0;
}

?? Ленивые вычисления (short-circuit evaluation)

Оператор &&: если левая часть false — правая не вычисляется.
Оператор ||: если левая часть true — правая не вычисляется.
Это фундаментальный механизм защиты от ошибок (деление на ноль, нулевой указатель).

Типичные паттерны использования

C++ 
#include <iostream>
#include <string>

// Функции, возвращающие bool — семантически чистее int
bool isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; ++i)
        if (n % i == 0) return false;
    return true;
}

bool isValidAge(int age) {
    return age >= 0 && age <= 150;
}

int main() {
    // Флаги-bool для управления состоянием
    bool found{false};
    bool isRunning{true};

    // Цикл с bool-флагом
    int arr[] = {1, 5, 3, 7, 2};
    for (int x : arr) {
        if (x == 3) { found = true; break; }
    }

    std::cout << std::boolalpha;
    std::cout << "found: "   << found       << '\n';
    std::cout << "isPrime(7): " << isPrime(7) << '\n'; // true
    std::cout << "isPrime(9): " << isPrime(9) << '\n'; // false

    return 0;
}
? Ключевые выводы

• bool принимает только true (1) и false (0).
• Любое ненулевое значение > true; ноль и nullptr > false.
• std::boolalpha переключает вывод на "true"/"false".
• Логические операторы && и || — ленивые: правая часть может не вычисляться.
• Результат сравнений (==, < и др.) всегда bool.
