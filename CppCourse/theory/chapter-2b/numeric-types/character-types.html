Символьные типы

Символьные типы хранят отдельные символы и являются строительным материалом для строк. C++ унаследовал char от C и добавил богатое семейство Unicode-ориентированных типов, чтобы корректно работать с любым языком мира.

Тип char — базовый символьный тип

[basic.fundamental] §6.7.1 — ISO/IEC 14882 Type char is a distinct type that has an implementation-defined choice of signed or unsigned representation. Objects declared as characters (char) shall be large enough to store any member of the implementation's basic character set.
Размер sizeof(char) == 1 (гарантия стандарта)
Знаковость implementation-defined (зависит от платформы)
Диапазон (signed) ?128 … 127
Диапазон (unsigned) 0 … 255
? Знаковость char не определена стандартом

Простой char может быть знаковым или беззнаковым — это решает компилятор. На x86 обычно signed, на некоторых ARM — unsigned. Когда знаковость важна, используйте явно signed char или unsigned char.

C++ 
#include <iostream>

int main() {
    char c{'A'};

    // char хранит числовой код ASCII
    std::cout << "char: " << c           << '\n'; // A
    std::cout << "код:  " << (int)c      << '\n'; // 65

    // Арифметика над char — это арифметика над ASCII-кодами
    char next{c + 1};
    std::cout << "A+1:  " << next        << '\n'; // B

    // Разница кодов
    char z{'z'}, a{'a'};
    std::cout << "'z'-'a' = " << (z - a) << '\n'; // 25

    // sizeof гарантированно равен 1
    std::cout << "sizeof(char) = " << sizeof(char) << '\n';

    return 0;
}

signed char, unsigned char и char — три разных типа

В C++ char, signed char и unsigned char — это три различных типа, даже несмотря на то что занимают одинаковое количество памяти:

Тип Размер Диапазон Назначение
signed char 1 байт (8 бит) ?128 … 127 Маленькое знаковое целое число
unsigned char 1 байт (8 бит) 0 … 255 Байт данных (пиксель, сетевой пакет)
char 1 байт (8 бит) signed или unsigned — зависит от компилятора Хранение числового кода символа ASCII

Несмотря на то что char может иметь тот же диапазон, что и signed char, эти типы не эквивалентны. char предназначен для хранения числового кода символа, тогда как signed char — для хранения маленького числа. На разных компиляторах char может вести себя как знаковый или беззнаковый, поэтому при важности знаковости всегда используйте явную форму.

C++ 
#include <iostream>

int main() {
    char          c  {'A'};   // для символов (предпочтительно)
    signed char   sc {-50};   // маленькое знаковое целое
    unsigned char uc {200};   // маленькое беззнаковое целое

    std::cout << "char    'A' = " << c       << " (код: " << (int)c  << ")\n";
    std::cout << "signed  sc  = " << (int)sc << '\n';
    std::cout << "unsigned uc = " << (int)uc << '\n';

    // Инициализация через код — допустима, но не предпочтительна
    char ch_by_code{ 97 };       // инициализация числом 97
    char ch_by_literal{ 'a' };  // предпочтительный способ
    // Обе переменные хранят одно значение, но читаемость второй выше

    return 0;
}
? Не путайте числовое и символьное значение

char ch{5}; — хранит число 5 (управляющий символ ENQ, непечатаемый).
char ch{'5'}; — хранит код символа '5', то есть число 53.
Это принципиально разные вещи!

Кодировка ASCII

ASCII (American Standard Code for Information Interchange) — 7-битная кодировка, определяющая 128 символов (коды 0–127). Коды 0–31 и 127 — непечатаемые управляющие символы. Коды 32–126 — печатаемые знаки.

Диапазон кодов Категория Примеры
0–31, 127Управляющие (непечатаемые)NUL (0), LF (10), CR (13), ESC (27)
32Пробел' '
48–57Цифры'0'=48, '1'=49 … '9'=57
65–90Заглавные буквы'A'=65, 'B'=66 … 'Z'=90
97–122Строчные буквы'a'=97, 'b'=98 … 'z'=122
?? Полезные соотношения в ASCII

• Разница между заглавной и строчной буквой одинакова: 'a' - 'A' == 32.
• Цифра '0' имеет код 48, поэтому '5' - '0' == 5.
• Для преобразования регистра: ch | 32 > строчная, ch & ~32 > заглавная (только для ASCII-букв).

C++ 
#include <iostream>

int main() {
    // Вывод ASCII-кода символа
    char ch{'A'};
    std::cout << "Символ: " << ch << ", код: " << (int)ch << '\n';

    // Соотношения
    std::cout << "'a' - 'A' = " << ('a' - 'A') << '\n';  // 32
    std::cout << "'5' - '0' = " << ('5' - '0') << '\n';  // 5 (числовое значение цифры)

    // Перевод цифры в число
    char digit{'7'};
    int  value{digit - '0'};  // 7
    std::cout << "Цифра '" << digit << "' как число: " << value << '\n';

    // Вывод всей таблицы печатаемых символов ASCII
    for (int code = 32; code <= 126; ++code)
        std::cout << (char)code;
    std::cout << '\n';

    return 0;
}

Ввод символов

std::cin >> пропускает ведущие пробелы и извлекает ровно один символ. Для извлечения пробельных символов используйте std::cin.get(ch):

C++ 
#include <iostream>

int main() {
    std::cout << "Введите символ: ";
    char ch{};
    std::cin >> ch;   // игнорирует пробелы перед символом
    std::cout << "Вы ввели: " << ch << '\n';

    // Если пользователь вводит "abc", то:
    // ch = 'a', а "bc" остаётся в буфере cin

    // Для чтения пробелов используйте get():
    char ch2{};
    std::cin.get(ch2); // извлекает следующий символ из буфера (может быть пробелом!)

    return 0;
}

Escape-последовательности

Некоторые символы нельзя вставить напрямую в код (перевод строки, кавычки, нулевой байт). Для них используются escape-последовательности — специальные комбинации символов, начинающиеся с \:

ИмяПоследовательностьЗначение
Alert (звонок)\aЗвуковой сигнал (BEL, код 7)
Backspace\bВозврат на один символ (код 8)
Form feed\fПеревод страницы (код 12)
Newline\nПеревод строки (LF, код 10)
Carriage return\rВозврат каретки (CR, код 13)
Horizontal tab\tГоризонтальная табуляция (код 9)
Vertical tab\vВертикальная табуляция (код 11)
Single quote\'Одинарная кавычка (код 39)
Double quote\"Двойная кавычка (код 34)
Backslash\\Обратный слэш (код 92)
Question mark\?Знак вопроса (устарело)
Null\0Нулевой символ — конец C-строки (код 0)
Hex-код\xNNСимвол с кодом NN (шестнадцатеричный)
Oct-код\NNNСимвол с кодом NNN (восьмеричный)
C++ 
#include <iostream>

int main() {
    std::cout << "Строка 1\nСтрока 2\n";          // \n — новая строка
    std::cout << "Столбец1\tСтолбец2\n";           // \t — табуляция
    std::cout << "Путь: C:\\Users\\Vasya\\\n";     // \\ — буквальный слэш
    std::cout << "Он сказал: \"Привет!\"\n";       // \" — кавычка внутри строки
    std::cout << "Null: \0скрыт\n";               // \0 — обрывает C-строку

    // Шестнадцатеричные escape-последовательности
    char ch{'\x41'}; // = 'A' (0x41 = 65 в десятичной)
    std::cout << ch << '\n';

    return 0;
}

Unicode-символьные типы

Для работы с интернациональным текстом C++ предоставляет специальные типы. Все они появились для явной семантики кодировки:

Тип Введён Кодировка Размер Литерал
wchar_t C++98 UTF-16 (Win) / UTF-32 (Linux) 2 б (Win) / 4 б (Linux) L'Я'
char16_t C++11 UTF-16 2 байта u'€'
char32_t C++11 UTF-32 4 байта U'??'
char8_t C++20 UTF-8 (явная семантика) 1 байт u8'A'
C++ 
#include <iostream>

int main() {
    char     c  {'A'};           // ASCII / UTF-8 байт
    wchar_t  wc {L'Я'};         // Широкий символ
    char16_t c16{u'\u20AC'};    // € (U+20AC) в UTF-16
    char32_t c32{U'\U0001F680'}; // ?? в UTF-32

    std::cout << "sizeof(char)     = " << sizeof(char)     << '\n';
    std::cout << "sizeof(wchar_t)  = " << sizeof(wchar_t)  << '\n';
    std::cout << "sizeof(char16_t) = " << sizeof(char16_t) << '\n';
    std::cout << "sizeof(char32_t) = " << sizeof(char32_t) << '\n';

    // char16_t хранит числовой код Unicode
    std::cout << "€ code point: " << (int)c16 << '\n'; // 8364

    return 0;
}

?? Почему wchar_t непортабелен

wchar_t имеет разный размер на Windows (2 байта, UTF-16) и Linux (4 байта, UTF-32). Для переносимого Unicode-кода предпочитайте char32_t или char8_t (C++20) с явной семантикой.

Арифметика над символами

Поскольку char — это числовой тип (1 байт), над ним можно выполнять арифметические операции. Это используется для работы с ASCII:

C++ 
#include <iostream>
#include <cctype>

int main() {
    // Проверки символов через <cctype>
    char c{'A'};
    std::cout << std::boolalpha;
    std::cout << "isalpha('A'): " << (bool)std::isalpha(c) << '\n'; // true
    std::cout << "isupper('A'): " << (bool)std::isupper(c) << '\n'; // true
    std::cout << "isdigit('A'): " << (bool)std::isdigit(c) << '\n'; // false

    // Конвертация регистра
    char lower{std::tolower('A')}; // 'a'
    char upper{std::toupper('z')}; // 'Z'
    std::cout << "tolower('A') = " << lower << '\n';
    std::cout << "toupper('z') = " << upper << '\n';

    // Перебор алфавита
    for (char ch = 'a'; ch <= 'z'; ++ch)
        std::cout << ch;
    std::cout << '\n';

    return 0;
}
