    Пространства имён

    Когда программа растёт и использует сторонние библиотеки, легко получить конфликт имён — две функции с одинаковым именем из разных источников. Пространства имён (namespaces) решают эту проблему, группируя имена в отдельные области.

    Объявление и использование

    C++ — namespace 
    // загрузка...

    using-declaration и using-directive

    Постоянно писать math:: утомительно. Есть два способа сократить запись, но у каждого свои риски.

    C++ — using 
    // загрузка...
    Никогда не пиши using namespace в заголовочных файлах

    using namespace std; в .h файле «заражает» все файлы, которые его включают. Это может вызвать неожиданные конфликты имён. В .cpp файлах — допустимо, но лучше использовать явную квалификацию.
