    Многомерные массивы

    Массив может содержать другие массивы в качестве элементов. Такие массивы называются многомерными. Самый распространённый случай — двумерный массив, который удобно представлять как таблицу.

    Объявление двумерного массива

    C++ — двумерный массив 
    // int имя[строки][столбцы]
int numbers[3][2]; // 3 строки, 2 столбца

// Инициализация
int nums[3][2]
{
    { 1, 2 },  // строка 0
    { 3, 4 },  // строка 1
    { 5, 6 }   // строка 2
};

// Нулевая инициализация
int zeros[3][2] {};

// Частичная инициализация
int partial[3][2] { {1, 2}, {}, {7} }; // {1,2}, {0,0}, {7,0}

// Первую размерность можно опустить
int auto_rows[][2] { {1,2}, {3,4}, {5,6} }; // 3 строки

    Визуализация как таблица

    Доступ к элементам

    C++ — чтение и запись элементов 
    #include <iostream>

int main()
{
    int nums[3][2] { {1,2}, {3,4}, {5,6} };

    int n = nums[1][0]; // строка 1, столбец 0 → 3
    std::cout << n << '\n'; // 3

    std::cout << nums[2][1] << '\n'; // 6

    nums[2][1] = 123; // изменяем элемент
    std::cout << nums[2][1] << '\n'; // 123

    return 0;
}

    Перебор вложенными циклами

    C++ — вложенные циклы 
    #include <iostream>

int main()
{
    const int rows = 3, cols = 2;
    int numbers[rows][cols] { {1,2}, {3,4}, {5,6} };

    // Вариант 1: индексы
    for (int i = 0; i < rows; ++i)
    {
        for (int j = 0; j < cols; ++j)
            std::cout << numbers[i][j] << '\t';
        std::cout << '\n';
    }

    // Вариант 2: range-for (внешний — по ссылке!)
    for (auto& row : numbers)
    {
        for (int val : row)
            std::cout << val << '\t';
        std::cout << '\n';
    }

    return 0;
}
    ⚠️ Внешний цикл range-for — по ссылке

    При переборе многомерного массива через range-for внешняя переменная должна быть ссылкой (auto&), иначе подмассив распадётся в указатель.
