    C-style массивы

    Помимо std::vector, в C++ есть C-style массивы — низкоуровневые массивы, унаследованные из языка C. Их размер фиксируется на этапе компиляции и не может меняться. В современном C++ предпочтительнее std::vector или std::array, но C-style массивы встречаются в старом коде и в некоторых системных контекстах.

    Объявление и инициализация

    C++ — C-style массивы 
    // Объявление: тип имя[размер]
int numbers[4];          // 4 элемента, значения не определены!

// Нулевая инициализация
int zeros[4] {};         // {0, 0, 0, 0}

// Инициализация значениями
int nums[4] { 1, 2, 3, 4 };

// Размер выводится из инициализатора
int auto_size[] { 1, 2, 3, 4, 5, 6 }; // 6 элементов

// Частичная инициализация: остальные = 0
int partial[4] { 1, 2 }; // {1, 2, 0, 0}

// Константный массив
const int fixed[4] { 1, 2, 3, 4 };
// fixed[0] = 99; // ошибка!
    Размер должен быть константой

    Размер C-style массива должен быть константным выражением, известным на этапе компиляции. Переменная как размер — нестандартное расширение (VLA), которого следует избегать.

    Доступ к элементам

    C++ — индексация 
    #include <iostream>

int main()
{
    int numbers[4] { 1, 2, 3, 4 };

    int first = numbers[0];  // 1
    numbers[0] = 34;         // изменяем первый элемент
    std::cout << numbers[0] << '\n'; // 34

    // Выход за границы — неопределённое поведение!
    // int bad = numbers[4]; // UB: индекс 4 не существует

    return 0;
}

    Получение длины

    C++ — длина массива 
    #include <iostream>

int main()
{
    int numbers[] { 11, 12, 13, 14 };

    // Способ 1: sizeof (из C)
    int len1 = sizeof(numbers) / sizeof(numbers[0]); // 4

    // Способ 2: std::size() (C++17, предпочтительно)
    int len2 = std::size(numbers); // 4

    std::cout << len1 << ' ' << len2 << '\n'; // 4 4
    return 0;
}

    Перебор

    C++ — перебор C-style массива 
    #include <iostream>

int main()
{
    int numbers[] { 1, 2, 3, 4 };
    int n = std::size(numbers);

    // Обычный for
    for (int i = 0; i < n; ++i)
        std::cout << numbers[i] << ' ';
    std::cout << '\n';

    // Range-based for (работает с C-style массивами!)
    for (auto x : numbers)
        std::cout << x << ' ';
    std::cout << '\n';

    // Сумма элементов
    int sum {};
    for (int i {}; i < n; sum += numbers[i++]);
    std::cout << "Сумма: " << sum << '\n'; // 10

    return 0;
}

    C-style vs std::vector

    Свойство C-style int arr[] std::vector
    Размер Фиксирован на этапе компиляции Динамический
    Метод size() Нет (нужен std::size()) Есть
    Передача в функцию Распадается в указатель (теряет размер) Передаётся по ссылке с размером
    Когда использовать Редко: legacy-код, низкоуровневые задачи В большинстве случаев
    Лучшая практика

    В современном C++ предпочитайте std::vector (динамический) или std::array (фиксированный) вместо C-style массивов.
