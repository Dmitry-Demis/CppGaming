class cppQuest.Server.Models.AppDbContext ( DbContextOptions< AppDbContext > options)

Основной DB-контекст приложения. Содержит конфигурацию сущностей и сиды для начального наполнения.

Конфигурирует модель данных — делегирует на небольшие метод-компоненты, чтобы код был читабельнее и каждая часть была ответственна за свою сущность.
