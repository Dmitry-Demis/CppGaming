if constexpr C++17

if constexpr — конструкция ветвления, вычисляемая на этапе компиляции. Ветка, условие которой ложно, полностью исключается из скомпилированного кода — компилятор даже не проверяет её синтаксис в контексте конкретного типа.

Это принципиально отличает if constexpr от обычного if с constexpr-условием: обычный if компилирует обе ветки, if constexpr — только ту, которая выбрана.

Без if constexpr шаблонная функция с кодом, специфичным для одного типа, не скомпилировалась бы для других типов. if constexpr решает эту проблему, исключая нерелевантные ветки.
