Безопасное сравнение — std::cmp_*C++20

C++20 добавил набор функций в <utility> для безопасного сравнения целых чисел разной знаковости без рисков UB или неожиданных результатов:

ФункцияЭквивалентБезопасна?
std::cmp_equal(a, b)a == b✅ да
std::cmp_less(a, b)a < b✅ да
std::cmp_greater(a, b)a > b✅ да
std::in_range<T>(n)проверяет, вмещается ли n в тип T✅ да
