Атрибут C++17 [[fallthrough]]

Атрибут [[fallthrough]] явно документирует намеренный переход из одного case в следующий без break. Компилятор не выдаёт предупреждение там, где стоит этот атрибут.

Атрибут размещается в конце тела case перед следующей меткой:

C++17 — [[fallthrough]] 
// загрузка...
Инициализатор в switch C++17

Начиная с C++17, в заголовке switch можно объявить переменную по аналогии с if (init; condition): 

switch (int x {getValue()}; x % 2)
{
    case 0: std::cout << "чётное\n"; break;
    case 1: std::cout << "нечётное\n"; break;
}

Переменная x видна только внутри блока switch. Это сужает область видимости и предотвращает случайное использование переменной за пределами конструкции.
